## Number Sets

| Symbol | Set | Contains |
|--------|-----|---------|
| ℕ | Natural | 1, 2, 3, 4, … |
| ℤ | Integers | …, −2, −1, 0, 1, 2, … |
| ℚ | Rational | Any p/q, p,q ∈ ℤ, q ≠ 0 |
| ℝ | Real | All rational + irrational |

Hierarchy: **ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ**

## HCF and LCM (via Prime Factorisation)

```
HCF = product of COMMON prime factors at their LOWEST power
LCM = product of ALL prime factors at their HIGHEST power

HCF(a, b) × LCM(a, b) = a × b
```

## Divisibility Rules

| Divisible by | Rule |
|-------------|------|
| 2 | Last digit even |
| 3 | Sum of digits divisible by 3 |
| 4 | Last two digits divisible by 4 |
| 5 | Ends in 0 or 5 |
| 6 | Divisible by both 2 and 3 |
| 8 | Last three digits divisible by 8 |
| 9 | Sum of digits divisible by 9 |
| 10 | Ends in 0 |

## Index Laws

| Law | Rule |
|-----|------|
| Multiply same base | aᵐ × aⁿ = aᵐ⁺ⁿ |
| Divide same base | aᵐ ÷ aⁿ = aᵐ⁻ⁿ |
| Power of a power | (aᵐ)ⁿ = aᵐⁿ |
| Zero index | a⁰ = 1 (a ≠ 0) |
| Negative index | a⁻ⁿ = 1/aⁿ |
| Fractional index | a^(1/n) = ⁿ√a |
| Fractional index | a^(m/n) = (ⁿ√a)ᵐ |

## Signed Number Rules

```
+ × + = +    − × − = +
+ × − = −    − × + = −

Adding negative  →  subtract
Subtracting negative  →  add
```
