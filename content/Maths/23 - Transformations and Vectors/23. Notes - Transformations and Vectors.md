## Transformations

### 1. Translation

Every point moves the **same distance in the same direction**.

Described by a **column vector**: $\begin{pmatrix} x \\ y \end{pmatrix}$

where x = horizontal shift (+ right, − left) and y = vertical shift (+ up, − down)

> [!example] Translate triangle by $\begin{pmatrix} 3 \\ -2 \end{pmatrix}$: move 3 right, 2 down.

- Shape, size, and orientation are **unchanged** (congruent)

---

### 2. Reflection

Every point maps to its **mirror image** across a line (the **mirror line**).

Common mirror lines:
- x-axis: (x, y) → (x, −y)
- y-axis: (x, y) → (−x, y)
- y = x: (x, y) → (y, x)
- y = −x: (x, y) → (−y, −x)
- y = c (horizontal): reflect in that horizontal line
- x = c (vertical): reflect in that vertical line

> [!example] Reflect (3, 5) in the y-axis → **(−3, 5)**

> [!example] Reflect (2, 4) in the line y = x → **(4, 2)**

- Shape and size are **unchanged** (congruent)
- Orientation is **reversed** (mirror image)

---

### 3. Rotation

Every point turns through an **angle** about a fixed **centre of rotation**.

Must specify:
1. Centre of rotation (x, y)
2. Angle (e.g. 90°, 180°, 270°)
3. Direction (clockwise or anticlockwise)

> [!tip] 180° rotation is the same whether clockwise or anticlockwise.

#### Rotation Rules (about origin):

| Rotation | Rule |
|----------|------|
| 90° anticlockwise | (x, y) → (−y, x) |
| 90° clockwise | (x, y) → (y, −x) |
| 180° | (x, y) → (−x, −y) |
| 270° anticlockwise | (x, y) → (y, −x) |

> [!example] Rotate (3, 2) by 90° anticlockwise about origin → **(−2, 3)**

- Shape and size are **unchanged** (congruent)

---

### 4. Enlargement

Every point is scaled by a **scale factor** from a **centre of enlargement**.

**Scale factor (k):**
- k > 1: larger image
- 0 < k < 1: smaller image (reduction)
- k < 0: image is on **opposite side** of centre and inverted

Finding the image point:
1. Draw a line from centre through each vertex
2. Multiply the distance from centre by k

> [!example] Enlarge triangle by scale factor 3 from centre (0, 0):
> Point (2, 1) → (6, 3)

> [!example] Enlargement with k = −2, centre (1, 1), point (3, 2):
> Vector from centre to point: (2, 1)
> Multiply by −2: (−4, −2)
> Image = (1 + (−4), 1 + (−2)) = **(−3, −1)**

> [!important] Area of enlarged shape = k² × area of original

---

## Describing Transformations

| Transformation | Must state |
|---------------|-----------|
| Translation | Column vector $\begin{pmatrix} x \\ y \end{pmatrix}$ |
| Reflection | Equation of mirror line |
| Rotation | Centre, angle, direction |
| Enlargement | Centre of enlargement, scale factor |

> [!warning] Always state the transformation type first, then the full description.

## Vectors

A **vector** has both **magnitude (size)** and **direction**.

### Notation
- Bold: **a** or with underline: <u>a</u>
- Column vector: $\begin{pmatrix} x \\ y \end{pmatrix}$ — x = horizontal, y = vertical component
- Arrow: $\overrightarrow{AB}$

### Magnitude (Length) of a Vector
$$|\mathbf{a}| = \sqrt{x^2 + y^2}$$

> [!example] $\mathbf{a} = \begin{pmatrix} 3 \\ 4 \end{pmatrix}$: |**a**| = √(9+16) = √25 = **5**

### Vector Addition and Subtraction
$$\begin{pmatrix} a \\ b \end{pmatrix} + \begin{pmatrix} c \\ d \end{pmatrix} = \begin{pmatrix} a+c \\ b+d \end{pmatrix}$$

> [!example] **a** + **b** = $\begin{pmatrix} 3 \\ 1 \end{pmatrix} + \begin{pmatrix} -1 \\ 4 \end{pmatrix} = \begin{pmatrix} 2 \\ 5 \end{pmatrix}$

### Scalar Multiplication
$$k \begin{pmatrix} a \\ b \end{pmatrix} = \begin{pmatrix} ka \\ kb \end{pmatrix}$$

Scales the magnitude by |k|; reverses direction if k < 0.

### Negative Vector
−**a** has the same magnitude as **a** but opposite direction.

---

## Vector Geometry

### Position Vectors
The position vector of point P is $\overrightarrow{OP}$ — vector from origin O to P.

### Finding Vectors Between Points
$$\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA} = \mathbf{b} - \mathbf{a}$$

> [!example] A = (2, 5), B = (7, 1).
> $\overrightarrow{AB} = \begin{pmatrix} 7-2 \\ 1-5 \end{pmatrix} = \begin{pmatrix} 5 \\ -4 \end{pmatrix}$

### Midpoint Using Vectors
Midpoint M of AB: $\overrightarrow{OM} = \frac{1}{2}(\mathbf{a} + \mathbf{b})$

### Parallel Vectors
Two vectors are **parallel** if one is a scalar multiple of the other.

**a** = k**b** → **a** ∥ **b**

> [!example] Show $\begin{pmatrix} 6 \\ -4 \end{pmatrix}$ is parallel to $\begin{pmatrix} -3 \\ 2 \end{pmatrix}$:
> $\begin{pmatrix} 6 \\ -4 \end{pmatrix} = -2 \begin{pmatrix} -3 \\ 2 \end{pmatrix}$ → **parallel** (scalar multiple −2)

### Collinear Points
Points are **collinear** (lie on the same line) if the vectors between them are parallel.

## Key Definitions

| Term | Definition |
|------|-----------|
| Translation | Sliding every point the same distance and direction |
| Reflection | Mirror image across a line |
| Rotation | Turning about a fixed centre through a given angle |
| Enlargement | Scaling from a centre by a scale factor |
| Congruent | Same shape and size (translations, reflections, rotations preserve this) |
| Similar | Same shape, different size (enlargement produces similar shapes) |
| Vector | Quantity with both magnitude and direction |
| Scalar | A quantity with only magnitude (a plain number) |
| Position vector | Vector from the origin to a point |
| Parallel vectors | Vectors with the same or opposite directions |
