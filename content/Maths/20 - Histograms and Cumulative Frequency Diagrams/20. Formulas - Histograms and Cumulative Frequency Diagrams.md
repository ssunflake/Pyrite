## Histograms

```
Frequency Density  =  Frequency / Class Width
Frequency          =  Frequency Density × Class Width  =  Area of bar
```

> y-axis = Frequency Density (NOT frequency)
> Area of bar = Frequency

---

## Cumulative Frequency

```
Cumulative frequency = running total of all frequencies up to that class
Plot at: UPPER boundary of each class
Start:   (lower boundary of first class, 0)
Shape:   smooth S-curve (ogive)
```

## Reading Off the Ogive

```
Median (Q2):          y = n/2
Lower quartile (Q1):  y = n/4
Upper quartile (Q3):  y = 3n/4
p-th percentile:      y = p × n / 100

IQR = Q3 − Q1
```

---

## Frequency Polygon

```
Plot frequency (or FD) at CLASS MIDPOINT
Join with straight lines
```
