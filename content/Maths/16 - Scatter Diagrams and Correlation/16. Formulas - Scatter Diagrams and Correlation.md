# 16. Formulas & Quick Reference — Scatter Diagrams and Correlation

[[16. Notes - Scatter Diagrams and Correlation|← Notes]] | [[00. Maths Index|Index]]

---

## Correlation Summary

```
Positive:   both variables increase together  →  points slope ↗
Negative:   one increases, one decreases      →  points slope ↘
None:       no pattern                        →  random scatter

Strong:   points close to a line
Weak:     points loosely spread around a line
```

## Mean Point

```
x̄  =  Σx / n       (mean of x-values)
ȳ  =  Σy / n       (mean of y-values)

Mean point (x̄, ȳ) must lie on the line of best fit.
```

---

## Prediction Types

```
Interpolation:   within data range    →  reliable
Extrapolation:   outside data range   →  unreliable
```

---

## Exam Description Template

```
"There is a [strong/weak] [positive/negative] correlation.
 As [x variable] increases, [y variable] tends to [increase/decrease]."
```

---

## Key Reminder

```
Correlation ≠ Causation
A third (lurking) variable may explain both trends.
```
