## Rounding Rules

```
Decimal places:    count from decimal point → look at next digit
Significant figures: count from first non-zero digit → look at next digit
  next digit ≥ 5  →  round up
  next digit < 5  →  leave (round down)
```

## Bounds

```
Given value rounded to unit u:
  Lower bound  =  value − u/2   (inclusive ≥)
  Upper bound  =  value + u/2   (exclusive <)

Max(a + b) = max a + max b       Min(a + b) = min a + min b
Max(a × b) = max a × max b       Min(a × b) = min a × min b
Max(a / b) = max a / min b       Min(a / b) = min a / max b
Max(a − b) = max a − min b       Min(a − b) = min a − max b
```

## Compound Measures

```
Speed    =  Distance / Time        D = S × T       T = D / S
Density  =  Mass / Volume          M = D × V       V = M / D
Pressure =  Force / Area           F = P × A       A = F / P
```

## Speed Unit Conversions

```
1 m/s  =  3.6 km/h
1 km/h =  1/3.6 m/s  ≈  0.278 m/s
```

---

## Key Unit Conversions

```
Length:   1 km = 1000 m = 100 000 cm = 1 000 000 mm
Mass:     1 tonne = 1000 kg = 1 000 000 g
Volume:   1 m³ = 1000 litres = 1 000 000 cm³;   1 litre = 1000 cm³
Area:     1 m² = 10 000 cm²;   1 cm² = 100 mm²
```

## Time

```
Convert to hours:   minutes ÷ 60
Convert to minutes: hours × 60
e.g. 2 h 45 min = 2 + 45/60 = 2.75 hours
```
