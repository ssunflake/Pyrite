## Graph Summary Table

| Equation | Curve | Key Feature |
|----------|-------|-------------|
| y = ax²+bx+c | Parabola | Vertex at x = −b/2a |
| y = ax³+… | Cubic | S-shape; up to 3 roots |
| y = k/x | Hyperbola | Asymptotes: x=0, y=0 |
| y = aˣ (a>1) | Exponential growth | y-int = 1; never hits x-axis |
| y = aˣ (0<a<1) | Exponential decay | y-int = 1; never hits x-axis |
| x²+y²=r² | Circle | Centre (0,0), radius r |
| (x−a)²+(y−b)²=r² | Circle | Centre (a,b), radius r |

## Graph Transformations

```
y = f(x) + a       →  shift UP by a
y = f(x) − a       →  shift DOWN by a
y = f(x + a)       →  shift LEFT by a    (a > 0)
y = f(x − a)       →  shift RIGHT by a   (a > 0)
y = af(x)          →  vertical stretch by a
y = f(ax)          →  horizontal stretch by 1/a
y = −f(x)          →  reflect in x-axis
y = f(−x)          →  reflect in y-axis
```

## Trig Graph Key Points

| Curve | Period | Amplitude | Range |
|-------|--------|-----------|-------|
| y = sin x | 360° | 1 | [−1, 1] |
| y = cos x | 360° | 1 | [−1, 1] |
| y = tan x | 180° | ∞ | ℝ |

```
sin 0° = 0,  sin 90° = 1,  sin 180° = 0,  sin 270° = −1
cos 0° = 1,  cos 90° = 0,  cos 180° = −1, cos 270° = 0
```

## Circle: Line Intersection

```
1. Express y (or x) from the line equation
2. Substitute into x² + y² = r²
3. Solve resulting quadratic
```
