## Functions

```
f(a)       →  substitute x = a into f(x)

Composite:
  fg(x)  =  f(g(x))   [apply g FIRST, then f]
  gf(x)  =  g(f(x))   [apply f FIRST, then g]

Inverse:
  1. Write y = f(x)
  2. Rearrange to x = ...
  3. Write f⁻¹(x) = ... (swap x and y labels)
```

---

## Algebraic Fractions

```
Simplify:    factorise top and bottom → cancel common factors

Multiply:    (a/b) × (c/d)  =  ac / bd     [cancel diagonally first]

Divide:      (a/b) ÷ (c/d)  =  (a/b) × (d/c)

Add/Sub:     find LCM of denominators → combine numerators
```

---

## Solving Equations with Fractions

```
1. Identify LCM of all denominators
2. Multiply every term by the LCM
3. Expand and simplify
4. Solve (may result in a quadratic)
```

---

## Inverse Function Properties

```
f(f⁻¹(x)) = x
f⁻¹(f(x)) = x
Domain of f⁻¹ = Range of f
Range  of f⁻¹ = Domain of f
```
