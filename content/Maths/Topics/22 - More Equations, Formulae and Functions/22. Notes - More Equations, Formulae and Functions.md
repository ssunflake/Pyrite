## Function Notation

A **function** f maps each input x to exactly one output f(x).

- f(x) = 3x + 1 means "multiply by 3, then add 1"
- f(2) means substitute x = 2

> [!example] f(x) = 2x¬≤ ‚ąí 3. Find f(4).
> f(4) = 2(16) ‚ąí 3 = **29**

> [!example] g(x) = 5 ‚ąí x. Find g(‚ąí3).
> g(‚ąí3) = 5 ‚ąí (‚ąí3) = **8**

## Domain and Range

- **Domain**: the set of allowed input values (x-values)
- **Range**: the set of possible output values (f(x)-values)

> [!example] f(x) = 1/x. Domain: all real x except x ‚Č† 0. Range: all real f(x) except f(x) ‚Č† 0.

> [!example] f(x) = ‚ąöx. Domain: x ‚Č• 0. Range: f(x) ‚Č• 0.

## Composite Functions

**fg(x)** means apply g first, then f.

fg(x) = f(g(x))

> [!warning] Order matters: fg ‚Č† gf in general.

> [!example] f(x) = 2x + 1, g(x) = x¬≤.
> fg(x) = f(g(x)) = f(x¬≤) = 2x¬≤ + 1
> gf(x) = g(f(x)) = g(2x+1) = (2x+1)¬≤

> [!example] f(x) = 3x ‚ąí 2, g(x) = x + 5. Find fg(4).
> g(4) = 9
> f(9) = 3(9) ‚ąí 2 = **25**

## Inverse Functions

The **inverse function** f‚ĀĽ¬Ļ(x) reverses f.

f(f‚ĀĽ¬Ļ(x)) = x and f‚ĀĽ¬Ļ(f(x)) = x

### Finding f‚ĀĽ¬Ļ(x):
1. Write y = f(x)
2. Rearrange to make x the subject
3. Replace x with f‚ĀĽ¬Ļ(x) and y with x

> [!example] f(x) = 3x + 2. Find f‚ĀĽ¬Ļ(x).
> y = 3x + 2 ‚Üí 3x = y ‚ąí 2 ‚Üí x = (y‚ąí2)/3
> **f‚ĀĽ¬Ļ(x) = (x‚ąí2)/3**

> [!example] f(x) = (2x+1)/(x‚ąí3). Find f‚ĀĽ¬Ļ(x).
> y(x‚ąí3) = 2x+1 ‚Üí xy ‚ąí 3y = 2x + 1 ‚Üí xy ‚ąí 2x = 3y+1
> x(y‚ąí2) = 3y+1 ‚Üí x = (3y+1)/(y‚ąí2)
> **f‚ĀĽ¬Ļ(x) = (3x+1)/(x‚ąí2)**

> [!tip] The domain of f‚ĀĽ¬Ļ is the range of f, and vice versa.

## Algebraic Fractions

### Simplifying
Factorise numerator and denominator, then cancel common factors.

> [!example] Simplify: (x¬≤ ‚ąí 4) / (x¬≤ + 4x + 4)
> = (x+2)(x‚ąí2) / (x+2)¬≤ = **(x‚ąí2)/(x+2)**

### Multiplying and Dividing
Multiply: (a/b) √ó (c/d) = ac/bd ‚ÄĒ factorise and cancel first

Divide: (a/b) √∑ (c/d) = (a/b) √ó (d/c)

> [!example] (x¬≤‚ąí9)/(x+2) √ó (2x+4)/(x+3)
> = (x+3)(x‚ąí3)/(x+2) √ó 2(x+2)/(x+3)
> = **2(x‚ąí3)**

### Adding and Subtracting
Find the LCM of denominators, then combine.

> [!example] 3/(x+1) + 2/(x‚ąí1)
> LCM = (x+1)(x‚ąí1)
> = [3(x‚ąí1) + 2(x+1)] / [(x+1)(x‚ąí1)]
> = [3x‚ąí3 + 2x+2] / (x¬≤‚ąí1)
> = **(5x‚ąí1) / (x¬≤‚ąí1)**

> [!example] 1/(x+2) ‚ąí 1/(x+3)
> = [(x+3) ‚ąí (x+2)] / [(x+2)(x+3)]
> = **1 / [(x+2)(x+3)]**

## Solving Equations with Algebraic Fractions

Multiply through by the LCM of denominators to clear all fractions.

> [!example] Solve: 2/(x‚ąí1) + 3/(x+2) = 1
> Multiply by (x‚ąí1)(x+2):
> 2(x+2) + 3(x‚ąí1) = (x‚ąí1)(x+2)
> 2x+4 + 3x‚ąí3 = x¬≤+x‚ąí2
> 5x+1 = x¬≤+x‚ąí2
> x¬≤ ‚ąí 4x ‚ąí 3 = 0
> x = (4 ¬Ī ‚ąö(16+12))/2 = (4 ¬Ī ‚ąö28)/2 = **2 ¬Ī ‚ąö7**

## Further Changing the Subject

> [!example] Make x the subject of: y = (ax + b)/(cx + d)
> y(cx+d) = ax+b
> cxy + dy = ax + b
> cxy ‚ąí ax = b ‚ąí dy
> x(cy ‚ąí a) = b ‚ąí dy
> **x = (b ‚ąí dy)/(cy ‚ąí a)**
