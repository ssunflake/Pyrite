## Fraction Operations

```
Add/Subtract:   common denominator → add/sub numerators
Multiply:       (a/b) × (c/d) = ac/bd      [cancel first if possible]
Divide:         (a/b) ÷ (c/d) = (a/b) × (d/c)
```

## Mixed Number → Improper

```
n(a/b)  =  (n×b + a) / b
```

---

## Percentage Formulas

```
% of amount         =  (% / 100) × amount

% change            =  [(new − original) / original] × 100

New value (inc/dec) =  original × multiplier
  Multiplier (n% increase) = 1 + n/100
  Multiplier (n% decrease) = 1 − n/100

Reverse % (find original):
  original = new value ÷ multiplier

Compound interest:
  A = P × (1 + r/100)ⁿ
  where P = principal, r = rate%, n = number of periods
```

---

## Standard Form

```
A × 10ⁿ    where  1 ≤ A < 10,  n ∈ ℤ

Multiply:   (A × 10ᵐ) × (B × 10ⁿ) = (A×B) × 10ᵐ⁺ⁿ  [adjust if A×B ≥ 10]
Divide:     (A × 10ᵐ) ÷ (B × 10ⁿ) = (A/B) × 10ᵐ⁻ⁿ  [adjust if A/B < 1]
Add/Sub:    convert to same power first, then add A values
```

## Key Multipliers to Know

| % change | Multiplier |
|----------|-----------|
| 5% inc | × 1.05 |
| 10% inc | × 1.10 |
| 15% inc | × 1.15 |
| 20% inc | × 1.20 |
| 25% inc | × 1.25 |
| 10% dec | × 0.90 |
| 20% dec | × 0.80 |
| 25% dec | × 0.75 |
| 50% dec | × 0.50 |

## Recurring Decimal → Fraction

```
For 0.abcabc…  (repeating block of 3):
  let x = 0.abcabc…
  1000x − x = abc
  x = abc / 999
```
