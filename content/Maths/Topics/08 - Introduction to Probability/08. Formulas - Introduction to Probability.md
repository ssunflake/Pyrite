# 08. Formulas & Quick Reference — Introduction to Probability

---

## Core Formulas

```
Basic probability:
  P(A) = (favourable outcomes) / (total outcomes)

Complement:
  P(A') = 1 − P(A)

AND — independent events:
  P(A and B) = P(A) × P(B)

OR — mutually exclusive:
  P(A or B) = P(A) + P(B)

OR — general (not mutually exclusive):
  P(A or B) = P(A) + P(B) − P(A and B)

Expected frequency:
  E = n × P(A)

Experimental probability:
  P(A) ≈ frequency / total trials
```

---

## Venn Diagram Notation

```
A ∩ B   →  A AND B  (intersection)
A ∪ B   →  A OR B   (union)
A'      →  NOT A    (complement)
ξ       →  universal set (everything)

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
```

---

## Mutually Exclusive vs Independent

```
Mutually exclusive:   P(A ∩ B) = 0
Independent:          P(A ∩ B) = P(A) × P(B)
```

---

## Key Rules

```
0 ≤ P(A) ≤ 1   (always)
P(certain)  = 1
P(impossible) = 0
Sum of all mutually exclusive outcomes = 1
```
