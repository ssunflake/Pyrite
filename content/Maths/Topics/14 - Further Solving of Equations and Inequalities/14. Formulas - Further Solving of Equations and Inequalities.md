## Solving Equations with Fractions

```
1. Find LCM of all denominators
2. Multiply every term by the LCM
3. Expand, collect, solve
```

## Simultaneous (Linear + Quadratic)

```
1. Express one variable from the linear equation
2. Substitute into the quadratic
3. Rearrange to standard form  ax² + bx + c = 0
4. Solve (factorise or formula)
5. Substitute back for the other variable
```

---

## Quadratic Inequalities

```
Solve equation first  →  find roots p and q  (p < q)

a > 0 (∪ shape):
  f(x) < 0  →  p < x < q          (between roots)
  f(x) > 0  →  x < p  or  x > q  (outside roots)

a < 0 (∩ shape):
  f(x) > 0  →  p < x < q
  f(x) < 0  →  x < p  or  x > q
```

---

## Graphical Inequalities

```
y ≤ mx + c  →  solid line;  shade BELOW
y < mx + c  →  dashed line; shade BELOW
y ≥ mx + c  →  solid line;  shade ABOVE
y > mx + c  →  dashed line; shade ABOVE

Test with (0,0) to confirm correct side
```

---

## Inequality Sign Flip Rule

```
Multiplying or dividing both sides by a NEGATIVE number
→ FLIP the inequality sign

e.g.  −2x > 6  →  x < −3
```
