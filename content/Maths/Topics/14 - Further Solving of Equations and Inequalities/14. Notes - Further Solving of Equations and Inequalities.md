## Harder Linear Equations

### Equations with Fractions
Multiply through by the LCM of all denominators to clear fractions.

> [!example] Solve: x/3 + (x−1)/2 = 4
> LCM(3, 2) = 6; multiply everything by 6:
> 2x + 3(x−1) = 24
> 2x + 3x − 3 = 24
> 5x = 27 → **x = 27/5**

> [!example] Solve: 3/(x+1) = 2/(x−2)
> Cross-multiply: 3(x−2) = 2(x+1)
> 3x − 6 = 2x + 2 → **x = 8**

### Equations with the Unknown in the Denominator
Multiply through to eliminate the denominator.

> [!example] Solve: 5/x − 3 = 2
> 5/x = 5 → **x = 1**

> [!example] Solve: 1/(x+1) + 1/(x+2) = 1/3
> Multiply through by 3(x+1)(x+2):
> 3(x+2) + 3(x+1) = (x+1)(x+2)
> 3x+6 + 3x+3 = x² + 3x + 2
> 6x + 9 = x² + 3x + 2
> x² − 3x − 7 = 0 → use quadratic formula

## Simultaneous Equations — Harder Cases

### Both Non-linear (or One Quadratic)
Substitute the linear into the quadratic.

> [!example] y = x + 1 and x² + y² = 25 (circle)
> Sub: x² + (x+1)² = 25
> x² + x² + 2x + 1 = 25
> 2x² + 2x − 24 = 0
> x² + x − 12 = 0 → (x+4)(x−3) = 0
> x = −4, y = −3 OR **x = 3, y = 4**

### Three Simultaneous Linear Equations
Use elimination twice — eliminate the same variable from two pairs of equations.

> [!example]
> (1): 2x + y + z = 7
> (2): x − y + 2z = 5
> (3): 3x + 2y − z = 8
>
> (1)+(2): 3x + 3z = 12 → x + z = 4 … (4)
> 2×(2)+(3): 5x + 3z = 18 … (5)
> (5)−(4)×3: 2x = 6 → x = 3, z = 1, y = 0

## Inequalities

### Linear Inequalities — Review
- Solve like an equation
- **Flip sign** when multiplying or dividing by a negative

> [!example] Solve: 2 − 3x ≤ 8
> −3x ≤ 6 → x ≥ −2 ← (flipped)

### Combined Inequalities
> [!example] Solve: −3 < 2x + 1 ≤ 9
> Subtract 1: −4 < 2x ≤ 8
> Divide by 2: **−2 < x ≤ 4**
> Integers: −1, 0, 1, 2, 3, 4

### Quadratic Inequalities
Solve the equation first, then determine the direction of the inequality using a sketch.

> [!example] Solve: x² − 5x + 6 < 0
> Roots: (x−2)(x−3) = 0 → x = 2 or x = 3
> Parabola opens upward (a > 0) → below x-axis BETWEEN roots
> Solution: **2 < x < 3**

> [!example] Solve: x² − x − 6 ≥ 0
> Roots: x = 3 or x = −2
> Parabola above x-axis OUTSIDE roots
> Solution: **x ≤ −2 or x ≥ 3**

> [!tip] For quadratic inequalities:
> - Parabola ∪ shape (a > 0):
>   - < 0 → between roots
>   - > 0 → outside roots
> - Parabola ∩ shape (a < 0): reverse the above

## Inequalities as Regions (Graphical)

On a graph, an inequality defines a **region**:
- Solid line (─) for ≤ or ≥ (boundary included)
- Dashed line (- -) for < or > (boundary excluded)
- Shade the **unwanted** region (convention) or indicate the **wanted** region

For a point test: substitute a point (often (0,0)) to check which side satisfies the inequality.

> [!example] Show the region y > 2x − 1
> Draw dashed line y = 2x − 1
> Test (0,0): 0 > −1 ✓ → shade the region containing (0,0)
> Label with R or state "region R is the unshaded area"

### Multiple Inequalities
The **feasible region** satisfies all inequalities simultaneously.
Shade unwanted regions one by one — the remaining unshaded area is the feasible region.

## Trial and Improvement

Used to find approximate solutions to equations that cannot be solved algebraically.

Method:
1. Identify that a solution exists in an interval
2. Try the midpoint value
3. Compare the result to the target
4. Narrow the interval and repeat to the required accuracy

> [!example] Solve x³ + x = 20 to 1 d.p.
> Try x=2: 8+2=10 (too small); x=3: 27+3=30 (too big) → solution between 2 and 3
> Try x=2.5: 15.625+2.5=18.125 (too small)
> Try x=2.7: 19.683+2.7=22.383 (too big)
> Try x=2.6: 17.576+2.6=20.176 (too big — just)
> Try x=2.55: 16.581+2.55=19.131 (too small)
> → solution between 2.55 and 2.6 → rounds to **x ≈ 2.6** (1 d.p.)

## Iteration

A repeated process of substituting into a rearranged formula to converge on a solution.

**Iterative formula:** xₙ₊₁ = f(xₙ)

> [!example] Use xₙ₊₁ = (xₙ³ + 5)/4, starting with x₀ = 2, to find a root of 4x = x³ + 5.
> x₁ = (8+5)/4 = 3.25
> x₂ = (34.328+5)/4 = 9.832 … (diverging)
> Note: not all iterative formulas converge — depends on the rearrangement.

## Key Definitions

| Term | Definition |
|------|-----------|
| Inequality | Statement comparing two expressions using <, >, ≤, ≥ |
| Linear inequality | Inequality with no powers higher than 1 |
| Quadratic inequality | Inequality involving x² |
| Feasible region | Region satisfying all constraints simultaneously |
| Simultaneous equations | System of equations with the same unknowns |
| Trial and improvement | Iterative narrowing of an interval to find an approximate root |
| Iteration | Applying a formula repeatedly to converge on a solution |
