## Straight Lines

```
Equation:         y = mx + c
Gradient:         m = (y₂−y₁) / (x₂−x₁)

Parallel lines:          m₁ = m₂
Perpendicular lines:     m₁ × m₂ = −1  →  m₂ = −1/m₁

Midpoint of (x₁,y₁) and (x₂,y₂):
  M = ((x₁+x₂)/2 , (y₁+y₂)/2)

Distance between two points:
  d = √[(x₂−x₁)² + (y₂−y₁)²]
```

## Quadratic Equations

```
Quadratic formula:
           −b ± √(b²−4ac)
  x  =  ─────────────────
                2a

Completing the square:
  x² + bx = (x + b/2)² − (b/2)²

Vertex form:
  y = a(x − h)² + k   →  vertex at (h, k)

Line of symmetry:
  x = −b / (2a)

Vertex:
  x = −b/(2a),  then substitute to find y
```

## Discriminant

```
b² − 4ac > 0  →  2 real roots      (crosses x-axis twice)
b² − 4ac = 0  →  1 repeated root   (touches x-axis)
b² − 4ac < 0  →  no real roots     (above/below x-axis)
```

## Parabola Summary

```
y = ax² + bx + c

a > 0  →  ∪ shape (minimum)
a < 0  →  ∩ shape (maximum)
y-intercept at (0, c)
Roots: solve ax²+bx+c = 0
Vertex at x = −b/2a
```
