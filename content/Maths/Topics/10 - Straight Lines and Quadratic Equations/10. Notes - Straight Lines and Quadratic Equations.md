## Straight Lines

### Gradient
**Gradient (m)** = rise / run = (y₂ − y₁) / (x₂ − x₁)

- Positive gradient → line slopes up left to right
- Negative gradient → line slopes down left to right
- Zero gradient → horizontal line
- Undefined gradient → vertical line

> [!example] Gradient between (2, 3) and (6, 11):
> m = (11−3)/(6−2) = 8/4 = **2**

### Equation of a Straight Line

**y = mx + c**
- m = gradient
- c = y-intercept (where line crosses y-axis)

> [!example] Find equation of line through (1, 5) with gradient 3:
> y = 3x + c; 5 = 3(1) + c → c = 2
> **y = 3x + 2**

> [!example] Find equation of line through (2, 1) and (4, 7):
> m = (7−1)/(4−2) = 3
> y = 3x + c; 1 = 3(2) + c → c = −5
> **y = 3x − 5**

### Rearranging to y = mx + c
> 2x + 3y = 12
> 3y = −2x + 12
> y = −⅔x + 4 → m = −2/3, c = 4

### x-intercept
Set y = 0 and solve for x.
> y = 3x − 5: 0 = 3x − 5 → x = 5/3

### Parallel Lines
Parallel lines have the **same gradient**.
> y = 2x + 1 and y = 2x − 3 are parallel (both m = 2)

### Perpendicular Lines
Perpendicular lines have gradients that are **negative reciprocals**:
m₁ × m₂ = −1 → m₂ = −1/m₁

> [!example] A line has gradient ⅔. A perpendicular line has gradient:
> m₂ = −1/(2/3) = **−3/2**

> [!example] Find equation of line perpendicular to y = 4x − 1 through (8, 2):
> m₁ = 4 → m₂ = −¼
> y = −¼x + c; 2 = −2 + c → c = 4
> **y = −¼x + 4**

### Midpoint and Length of a Line Segment

**Midpoint** = ((x₁+x₂)/2, (y₁+y₂)/2)

**Length** = √[(x₂−x₁)² + (y₂−y₁)²]

> [!example] A(1,3), B(5,7):
> Midpoint = (3, 5), Length = √(16+16) = **4√2**

## Quadratic Equations — Recap and Extension

Standard form: **ax² + bx + c = 0**

### Solving by Factorising
> x² + x − 12 = 0 → (x+4)(x−3) = 0 → **x = −4 or x = 3**

### Solving by Completing the Square
Write in form **(x + p)² = q**, then take square root.

> [!example] x² + 8x + 3 = 0
> (x+4)² − 16 + 3 = 0
> (x+4)² = 13
> x + 4 = ±√13
> **x = −4 ± √13**

Completing the square for ax² (divide through by a first):
> 2x² − 12x + 7 = 0 → x² − 6x + 7/2 = 0
> (x−3)² − 9 + 7/2 = 0 → (x−3)² = 11/2
> x = 3 ± √(11/2) = 3 ± **√22/2**

### Solving with the Quadratic Formula
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

> [!example] 3x² − 5x − 2 = 0
> Discriminant = 25 + 24 = 49
> x = (5 ± 7)/6 → **x = 2 or x = −1/3**

### The Discriminant b² − 4ac

| Value | Roots | Graph |
|-------|-------|-------|
| > 0 | 2 distinct real roots | Crosses x-axis twice |
| = 0 | 1 repeated real root | Touches x-axis once |
| < 0 | No real roots | Doesn't cross x-axis |

## Graphs of Quadratics

The graph of y = ax² + bx + c is a **parabola**.

| Feature | How to find |
|---------|------------|
| y-intercept | Set x = 0 → y = c |
| x-intercepts (roots) | Solve ax² + bx + c = 0 |
| Vertex (turning point) | x = −b/(2a); then find y |
| Line of symmetry | x = −b/(2a) |
| Opens up/down | a > 0 → ∪ (minimum); a < 0 → ∩ (maximum) |

> [!example] y = x² − 4x + 3
> y-intercept: y = 3
> Roots: (x−1)(x−3) = 0 → x = 1, x = 3
> Vertex: x = 4/2 = 2; y = 4−8+3 = −1 → **(2, −1)**
> Line of symmetry: **x = 2**

### Vertex Form (Completed Square Form)
y = a(x − h)² + k → vertex at **(h, k)**

> [!example] y = 2(x − 3)² + 5 → vertex at **(3, 5)**, minimum, opens upward

## Intersection of a Line and a Parabola

Substitute the linear equation into the quadratic, then solve.

> [!example] Find intersection of y = x + 2 and y = x²
> x² = x + 2 → x² − x − 2 = 0 → (x−2)(x+1) = 0
> x = 2 → y = 4 or x = −1 → y = 1
> Points: **(2, 4) and (−1, 1)**

## Key Definitions

| Term | Definition |
|------|-----------|
| Gradient | Steepness of a line; rise/run |
| y-intercept | Point where line crosses y-axis (x = 0) |
| x-intercept | Point where line crosses x-axis (y = 0) |
| Parallel | Lines with equal gradients |
| Perpendicular | Lines with gradients m₁ × m₂ = −1 |
| Parabola | The curved shape of a quadratic graph |
| Vertex | The minimum or maximum point of a parabola |
| Line of symmetry | Vertical line through the vertex of a parabola |
| Discriminant | b² − 4ac; determines the nature of quadratic roots |
| Completing the square | Rewriting a quadratic in (x+p)² form |
