## Scale Drawings

```
Scale = map length : real length
Real length  =  map length × scale denominator
Map length   =  real length ÷ scale denominator
```

## Bearings

```
Measured: clockwise from North, always 3 digits
Back bearing = bearing + 180°  (subtract 180° if result > 360°)
```

## Right-Angled Triangles (SOHCAHTOA)

```
sin θ = O/H     →  O = H sin θ     →  H = O/sin θ
cos θ = A/H     →  A = H cos θ     →  H = A/cos θ
tan θ = O/A     →  O = A tan θ     →  A = O/tan θ

Finding angle:
  θ = sin⁻¹(O/H)   θ = cos⁻¹(A/H)   θ = tan⁻¹(O/A)
```

## Exact Values

| θ | sin θ | cos θ | tan θ |
|---|-------|-------|-------|
| 30° | 1/2 | √3/2 | 1/√3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | 1/2 | √3 |

## Any Triangle

```
Sine rule:    a/sin A = b/sin B = c/sin C
              (use when: AAS, ASA, or SSA)

Cosine rule:  a² = b² + c² − 2bc·cos A       (finding side, given SAS)
              cos A = (b²+c²−a²) / (2bc)     (finding angle, given SSS)

Area:         Area = ½ab·sin C
              (where C = included angle between sides a and b)
```

---

## When to Use Which Rule

```
Right triangle          →  SOHCAHTOA
SAS or SSS              →  Cosine rule
AAS, ASA, or SSA        →  Sine rule
Area with 2 sides       →  ½ab sin C
```
