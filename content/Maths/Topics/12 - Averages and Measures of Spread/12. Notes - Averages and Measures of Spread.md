## The Three Averages

### Mean
Sum of all values divided by the number of values.

$$\bar{x} = \frac{\sum x}{n}$$

> [!example] Data: 4, 7, 2, 9, 3
> Mean = (4+7+2+9+3)/5 = 25/5 = **5**

> [!tip] The mean uses ALL values, so it is affected by extreme values (outliers).

### Median
The **middle value** when data is arranged in order.

- For n values: median is the ((n+1)/2)th value
- If n is even: median = mean of the two middle values

> [!example] Data: 3, 7, 8, 2, 5, 9, 1 → ordered: 1, 2, 3, **5**, 7, 8, 9
> n = 7 → median is (7+1)/2 = 4th value = **5**

> [!example] Data: 4, 6, 8, 10 → median = (6+8)/2 = **7**

> [!tip] The median is NOT affected by extreme values — use it for skewed data.

### Mode
The value that appears **most often**.

- Can have no mode, one mode (unimodal), or multiple modes (bimodal/multimodal)
- Only average that works for qualitative (categorical) data

> [!example] Data: 3, 5, 3, 7, 8, 3, 5 → mode = **3**

## Measures of Spread

### Range
**Range = Maximum − Minimum**

> [!example] Data: 2, 5, 8, 11, 3 → Range = 11 − 2 = **9**

A large range means data is spread out. A small range means data is clustered.

### Quartiles and Interquartile Range (IQR)

- **Q1 (Lower Quartile)**: median of the lower half
- **Q2**: median of whole dataset
- **Q3 (Upper Quartile)**: median of the upper half
- **IQR = Q3 − Q1**

The IQR covers the middle 50% of the data. It is NOT affected by outliers.

> [!example] Data (ordered): 2, 4, 5, 7, 9, 11, 13, 15
> n = 8; Q2 = (7+9)/2 = 8
> Lower half: 2, 4, 5, 7 → Q1 = (4+5)/2 = **4.5**
> Upper half: 9, 11, 13, 15 → Q3 = (11+13)/2 = **12**
> IQR = 12 − 4.5 = **7.5**

## Averages from a Frequency Table

| Value (x) | Frequency (f) | f × x |
|-----------|--------------|-------|
| 1 | 3 | 3 |
| 2 | 5 | 10 |
| 3 | 4 | 12 |
| 4 | 2 | 8 |
| **Total** | **Σf = 14** | **Σfx = 33** |

**Mean** = Σ(fx) / Σf = 33/14 ≈ **2.36**

**Median**: total frequency = 14 → median is between 7th and 8th values.
Cumulative frequencies: up to x=1: 3; up to x=2: 8 → 7th and 8th values are both **x = 2** → median = **2**

**Mode** = value with highest frequency = **2** (f=5)

## Estimated Mean from Grouped Data

Use the **midpoint** of each class to represent all values in that class.

| Class | Midpoint (x) | Frequency (f) | fx |
|-------|-------------|--------------|-----|
| 10 ≤ h < 20 | 15 | 4 | 60 |
| 20 ≤ h < 30 | 25 | 7 | 175 |
| 30 ≤ h < 40 | 35 | 9 | 315 |
| 40 ≤ h < 50 | 45 | 5 | 225 |
| **Total** | | **25** | **775** |

Estimated mean = 775/25 = **31**

> [!warning] This is an *estimate* because we don't know the exact values within each class.

### Modal Class
The class with the highest frequency. From the table above: **30 ≤ h < 40**

### Median Class
Find which class the (n/2)th or ((n+1)/2)th value falls in using cumulative frequencies.

## Box Plots (Box and Whisker Diagrams)

Represent the five-number summary: **Min, Q1, Median, Q3, Max**

```
|─────|──────|──|──────|──────|
Min  Q1   Median  Q3   Max
      └── IQR ──┘
```

Box plots allow easy comparison of two distributions.

## Comparing Distributions

When comparing two datasets, always comment on:
1. **Average** (mean or median) — which is higher/lower?
2. **Spread** (range or IQR) — which is more consistent/variable?

> [!example] Group A: median = 65, IQR = 20. Group B: median = 72, IQR = 35.
> "Group B has a higher median score (72 vs 65), but Group A is more consistent — lower IQR (20 vs 35) shows less variability in scores."

---

## Outliers

An **outlier** is a value that is much higher or lower than the rest.

Formal rule: an outlier is any value:
- below Q1 − 1.5 × IQR, OR
- above Q3 + 1.5 × IQR

## Key Definitions

| Term | Definition |
|------|-----------|
| Mean | Sum of values ÷ number of values |
| Median | Middle value in ordered data |
| Mode | Most frequent value |
| Range | Max − Min |
| Quartile | Q1, Q2 (median), Q3 divide ordered data into four equal parts |
| IQR | Interquartile range = Q3 − Q1; spread of middle 50% |
| Outlier | An extreme value far from the rest of the data |
| Modal class | Class with highest frequency in grouped data |
| Estimated mean | Mean calculated using midpoints of grouped data |
| Five-number summary | Min, Q1, Median, Q3, Max |
