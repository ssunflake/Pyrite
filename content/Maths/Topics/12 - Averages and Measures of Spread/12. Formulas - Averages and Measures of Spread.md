## Averages

```
Mean:    x̄  =  Σx / n         (raw data)
         x̄  =  Σ(fx) / Σf    (frequency table)
         x̄  ≈  Σ(fx) / Σf    (grouped — use midpoints, this is an ESTIMATE)

Median:  position  =  (n+1)/2 th value  (odd n)
                      mean of n/2 and n/2+1 th values (even n)

Mode:    value with highest frequency
         Modal class: class with highest frequency (grouped data)
```

---

## Spread

```
Range  =  Max − Min

IQR  =  Q3 − Q1

Outlier (below):  x < Q1 − 1.5 × IQR
Outlier (above):  x > Q3 + 1.5 × IQR
```

---

## Finding Quartiles

```
Order data first.
Q2 (median): middle value
Q1: median of lower half (exclude Q2 if n odd)
Q3: median of upper half (exclude Q2 if n odd)
```

## Comparison Phrases

```
Average:  "Group X has a higher/lower mean/median, suggesting..."
Spread:   "Group X has a smaller/larger IQR/range, meaning more/less consistent..."
```

---

## Five-Number Summary

```
Min  |  Q1  |  Median  |  Q3  |  Max
	    └────── IQR ──────┘
```
