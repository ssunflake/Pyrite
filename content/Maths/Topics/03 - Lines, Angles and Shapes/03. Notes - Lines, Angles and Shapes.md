| Name | Size |
|------|------|
| Acute | 0° < θ < 90° |
| Right | θ = 90° |
| Obtuse | 90° < θ < 180° |
| Straight | θ = 180° |
| Reflex | 180° < θ < 360° |
| Revolution | θ = 360° |

## Angle Rules

### On a Straight Line
Angles on a straight line add to **180°**

### Around a Point
Angles around a point add to **360°**

### Vertically Opposite Angles
When two straight lines cross, vertically opposite angles are **equal**.

### Parallel Lines (Transversal)
When a line (transversal) crosses two parallel lines:

| Pair | Relationship |
|------|-------------|
| Corresponding angles (F-shape) | **Equal** |
| Alternate angles (Z-shape) | **Equal** |
| Co-interior / Allied angles (C-shape) | **Add to 180°** |

> [!tip] Memory Aid
> - **F** for Corresponding (like an F-shape)
> - **Z** for Alternate (like a Z)
> - **C** for Co-interior (like a C)

## Triangles

### Angle Sum
The angles in any triangle add to **180°**.

### Types by Angle

| Type | Property |
|------|---------|
| Acute-angled | All angles < 90° |
| Right-angled | One angle = 90° |
| Obtuse-angled | One angle > 90° |

### Types by Side

| Type | Property |
|------|---------|
| Scalene | No equal sides, no equal angles |
| Isosceles | Two equal sides, two equal base angles |
| Equilateral | All sides equal, all angles = 60° |

### Exterior Angle of a Triangle
An exterior angle = sum of the **two non-adjacent interior angles**.

> [!example] A triangle has angles 40° and 65°. Find the exterior angle at the third vertex.
> Exterior angle = 40° + 65° = **105°**

## Quadrilaterals

### Angle Sum = 360°

| Shape | Properties |
|-------|-----------|
| Square | 4 equal sides, 4 right angles, diagonals equal and bisect at 90° |
| Rectangle | 4 right angles, opposite sides equal, diagonals equal |
| Rhombus | 4 equal sides, opposite angles equal, diagonals bisect at 90° |
| Parallelogram | Opposite sides equal and parallel, opposite angles equal |
| Trapezium | One pair of parallel sides |
| Isosceles trapezium | One pair of parallel sides, non-parallel sides equal, base angles equal |
| Kite | Two pairs of adjacent equal sides, one pair of equal angles, one diagonal bisects the other at 90° |

## Polygons

### Interior Angle Sum
Sum of interior angles = **(n − 2) × 180°** where n = number of sides

### Each Interior Angle (regular polygon)
= (n − 2) × 180° / n

### Each Exterior Angle (regular polygon)
= 360° / n

### Interior + Exterior = 180° (at each vertex)

> [!example] Regular hexagon (n = 6)
> Interior angle sum = (6−2) × 180° = **720°**
> Each interior angle = 720°/6 = **120°**
> Each exterior angle = 360°/6 = **60°**

| Polygon | n | Int. sum | Int. angle (regular) |
|---------|---|---------|----------------------|
| Triangle | 3 | 180° | 60° |
| Quadrilateral | 4 | 360° | 90° |
| Pentagon | 5 | 540° | 108° |
| Hexagon | 6 | 720° | 120° |
| Heptagon | 7 | 900° | 128.6° |
| Octagon | 8 | 1080° | 135° |
| Nonagon | 9 | 1260° | 140° |
| Decagon | 10 | 1440° | 144° |

## Circle Theorems

> [!important] These are examinable — learn them all!

| Theorem | Statement |
|---------|----------|
| **Angle at centre** | Angle at centre = 2 × angle at circumference (same arc) |
| **Angle in semicircle** | Angle in a semicircle = 90° |
| **Angles in same segment** | Angles in the same segment are equal |
| **Cyclic quadrilateral** | Opposite angles of a cyclic quadrilateral add to 180° |
| **Tangent-radius** | A tangent meets the radius at 90° |
| **Two tangents from external point** | Both tangents equal in length; line to centre bisects the angle |
| **Alternate segment** | Angle between tangent and chord = angle in alternate segment |
| **Chord bisector** | Perpendicular from centre to chord bisects the chord |

> [!example] Angle at centre
> If arc AB subtends angle 70° at circumference, it subtends **140°** at the centre.

## Congruence vs Similarity

| | Congruent | Similar |
|-|-----------|---------|
| Shape | Same | Same |
| Size | Same | Different (proportional) |
| Tests | SSS, SAS, AAS, RHS | AA, SAS, SSS (ratios) |

### Congruence Conditions (SSS / SAS / AAS / RHS)
- **SSS** — three sides equal
- **SAS** — two sides and the included angle equal
- **AAS** — two angles and a corresponding side equal
- **RHS** — right angle, hypotenuse, one side equal

## Key Definitions

| Term | Definition |
|------|-----------|
| Transversal | A line cutting across two or more parallel lines |
| Corresponding angles | Equal angles on same side of transversal (F-shape) |
| Alternate angles | Equal angles on opposite sides (Z-shape) |
| Co-interior angles | Supplementary angles on same side of transversal (C-shape) |
| Regular polygon | All sides equal and all angles equal |
| Cyclic quadrilateral | All four vertices lie on a circle |
| Tangent | A line touching a circle at exactly one point |
| Chord | A straight line joining two points on a circle |
| Arc | Part of the circumference |
| Sector | Region between two radii and an arc (pizza slice) |
| Segment | Region between a chord and an arc |
