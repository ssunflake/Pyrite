## Angle Rules

```
Straight line:          angles sum = 180°
Around a point:         angles sum = 360°
Triangle:               angles sum = 180°
Quadrilateral:          angles sum = 360°
Vertically opposite:    equal
```

## Parallel Lines (Transversal)

```
Corresponding (F):   equal
Alternate (Z):       equal
Co-interior (C):     sum = 180°
```

---

## Polygons

```
Interior angle sum  =  (n − 2) × 180°
Each interior angle =  (n − 2) × 180° / n   [regular]
Each exterior angle =  360° / n               [regular]
Interior + Exterior =  180°   [at each vertex]
```

---

## Exterior Angle of Triangle

```
Exterior angle = sum of two non-adjacent interior angles
```

## Circle Theorems Summary

| # | Theorem |
|---|---------|
| 1 | Angle at centre = 2 × angle at circumference |
| 2 | Angle in semicircle = 90° |
| 3 | Angles in same segment are equal |
| 4 | Opposite angles of cyclic quadrilateral = 180° |
| 5 | Tangent ⊥ radius at point of contact |
| 6 | Two tangents from external point are equal length |
| 7 | Alternate segment theorem: tangent-chord angle = angle in alternate segment |
| 8 | Perpendicular from centre bisects chord |

## Congruence Tests

```
SSS  —  3 sides equal
SAS  —  2 sides + included angle equal
AAS  —  2 angles + corresponding side equal
RHS  —  right angle + hypotenuse + one side
```
