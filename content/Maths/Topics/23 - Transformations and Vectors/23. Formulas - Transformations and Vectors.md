## Transformations Summary

| Type | Describe with | Preserves |
|------|--------------|-----------|
| Translation | Column vector | Size, shape, orientation |
| Reflection | Mirror line equation | Size, shape (not orientation) |
| Rotation | Centre, angle, direction | Size, shape |
| Enlargement | Centre, scale factor k | Shape (not size unless k=1) |

```
Enlargement area:    Area(image) = k² × Area(original)
```

## Rotation Rules (about origin)

```
90° anticlockwise:   (x, y) → (−y, x)
90° clockwise:       (x, y) → (y, −x)
180°:                (x, y) → (−x, −y)
```

## Common Reflection Rules

```
x-axis:   (x, y) → (x, −y)
y-axis:   (x, y) → (−x, y)
y = x:    (x, y) → (y, x)
y = −x:   (x, y) → (−y, −x)
```

---

## Vectors

```
Magnitude:    |a| = √(x² + y²)

Add:     (a,b) + (c,d) = (a+c, b+d)
Sub:     (a,b) − (c,d) = (a−c, b−d)
Scale:   k(a,b) = (ka, kb)

AB vector:    AB = b − a   (position vectors)
Midpoint M:   OM = ½(a + b)

Parallel:     a = kb  for some scalar k
Collinear:    AB and AC are parallel (share point A)
```
