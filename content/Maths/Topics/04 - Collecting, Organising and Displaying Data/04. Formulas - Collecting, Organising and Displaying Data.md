## Key Formulas

```
Pie chart sector angle  =  (frequency / total) × 360°

Class midpoint          =  (lower boundary + upper boundary) / 2

Class width             =  upper boundary − lower boundary

Relative frequency      =  frequency / total frequency
```

## Data Types Decision Tree

```
Is it described in words?
  YES → Qualitative (categorical)
  NO  → Quantitative
        Can it only take whole/discrete values?
          YES → Discrete
          NO  → Continuous
```

## Sampling Methods Summary

| Method | How |
|--------|-----|
| Random | Lottery / random number generator |
| Systematic | Every nth item (n = population/sample size) |
| Stratified | Sample from each group ∝ group size |

```
Stratified sample size from group  =  (group size / total) × sample size
```
