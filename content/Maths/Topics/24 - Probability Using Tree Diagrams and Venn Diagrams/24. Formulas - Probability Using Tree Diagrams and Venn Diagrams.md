## Tree Diagrams

```
Multiply ALONG branches   →  probability of that combined outcome
Add ACROSS branches       →  total probability of an event

All end-branch probabilities must sum to 1.

Without replacement:
  Numerator decreases by 1 if same type drawn
  Denominator always decreases by 1
```

---

## Core Probability Formulas

```
Complement:          P(A') = 1 − P(A)

Addition (general):  P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Addition (mutually exclusive):
                     P(A ∪ B) = P(A) + P(B)     [since P(A∩B)=0]

Multiplication (independent):
                     P(A ∩ B) = P(A) × P(B)

Conditional:         P(A | B)  =  P(A ∩ B) / P(B)

Independence check:  P(A | B) = P(A)  →  A and B are independent
```

---

## Venn Diagram Region Counts

```
n(A only)   = n(A) − n(A ∩ B)
n(B only)   = n(B) − n(A ∩ B)
n(A ∪ B)    = n(A) + n(B) − n(A ∩ B)
n(neither)  = n(ξ) − n(A ∪ B)
```

---

## Three-Set Venn — Fill Order

```
1. A ∩ B ∩ C  (centre)
2. Each pairwise intersection minus centre
3. Each circle's solo region = total − all overlaps
4. Neither = n(ξ) − n(A ∪ B ∪ C)
```
