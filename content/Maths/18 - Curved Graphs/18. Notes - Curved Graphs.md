## Quadratic Graphs (Recap)

y = ax² + bx + c — a **parabola**

| Feature | How to find |
|---------|------------|
| Shape | a > 0 → ∪ (min); a < 0 → ∩ (max) |
| y-intercept | Set x = 0: y = c |
| x-intercepts (roots) | Solve ax²+bx+c = 0 |
| Vertex | x = −b/2a; substitute for y |
| Line of symmetry | x = −b/2a |

---

## Cubic Graphs

y = ax³ + bx² + cx + d

| Feature | Description |
|---------|-------------|
| Shape (a > 0) | Falls left, rises right — S-curve (↗) |
| Shape (a < 0) | Rises left, falls right — inverted S (↙) |
| y-intercept | y = d (when x = 0) |
| x-intercepts | Up to 3 real roots |
| Turning points | Up to 2 (a local max and a local min) |

> [!example] y = x³ − 3x² + 2 passes through (0, 2). It has a local max and min.

> [!tip] For sketching: find the y-intercept and any x-intercepts (by factorising if possible), then sketch the S-shape.

---

## Reciprocal Graphs

$$y = \frac{k}{x}$$  or equivalently  y = kx⁻¹

| Feature | Description |
|---------|-------------|
| Shape | Two curves (hyperbola) in opposite quadrants |
| Asymptotes | x = 0 (y-axis) and y = 0 (x-axis) — never touches these |
| k > 0 | Curves in 1st and 3rd quadrants |
| k < 0 | Curves in 2nd and 4th quadrants |

> [!example] y = 6/x: as x → 0⁺, y → +∞; as x → +∞, y → 0

---

## Exponential Graphs

$$y = a^x \quad (a > 0,\, a \neq 1)$$

| Feature | a > 1 (growth) | 0 < a < 1 (decay) |
|---------|---------------|-------------------|
| Shape | Increasing curve | Decreasing curve |
| y-intercept | Always (0, 1) | Always (0, 1) |
| Asymptote | y = 0 (x-axis, as x → −∞) | y = 0 (x-axis, as x → +∞) |
| Never negative | y > 0 always | y > 0 always |

> [!example] y = 2ˣ: y-intercept = 1; passes through (1,2), (2,4), (3,8); (−1, 0.5)

> [!example] y = (0.5)ˣ = (1/2)ˣ: y-intercept = 1; passes through (1, 0.5), (2, 0.25)

## Circle Graphs

$$x^2 + y^2 = r^2$$

This is a **circle** centred at the **origin** with radius r.

> [!example] x² + y² = 25 → circle, centre (0,0), radius 5

**General form:** (x − a)² + (y − b)² = r² — circle centred at **(a, b)** radius r

> [!example] (x − 3)² + (y + 2)² = 16 → centre **(3, −2)**, radius **4**

To find where a line intersects a circle: substitute line into circle equation and solve.

## Other Recognisable Curves

| Equation | Curve type |
|----------|-----------|
| y = x² | Parabola (upward) |
| y = −x² | Parabola (downward) |
| y = x³ | Cubic S-curve |
| y = k/x | Hyperbola (reciprocal) |
| y = aˣ | Exponential (growth or decay) |
| x² + y² = r² | Circle |
| y = sin x, y = cos x, y = tan x | Trigonometric curves |

## Trigonometric Graphs

### y = sin x

| Feature | Value |
|---------|-------|
| Period | 360° |
| Amplitude | 1 |
| Range | −1 ≤ y ≤ 1 |
| Key points | (0,0), (90°,1), (180°,0), (270°,−1), (360°,0) |

### y = cos x

| Feature | Value |
|---------|-------|
| Period | 360° |
| Amplitude | 1 |
| Range | −1 ≤ y ≤ 1 |
| Key points | (0°,1), (90°,0), (180°,−1), (270°,0), (360°,1) |

### y = tan x

| Feature | Value |
|---------|-------|
| Period | 180° |
| Range | All real numbers |
| Asymptotes | x = 90°, 270°, 450°, … |
| Key points | (0°,0), (45°,1), (135°,−1) |

## Graph Transformations

For a base function y = f(x):

| Transformation | Equation | Effect |
|---------------|----------|--------|
| Translation up/down | y = f(x) + a | Moves graph up (a > 0) or down (a < 0) |
| Translation left/right | y = f(x + a) | Moves graph left (a > 0) or right (a < 0) |
| Stretch vertically | y = af(x) | Stretch by factor a from x-axis |
| Stretch horizontally | y = f(ax) | Stretch by factor 1/a from y-axis |
| Reflection in x-axis | y = −f(x) | Flips graph upside-down |
| Reflection in y-axis | y = f(−x) | Flips graph left-right |

> [!warning] y = f(x + 2) moves LEFT (not right) — the +2 is inside the function.

> [!example] y = sin(x + 90°) is y = sin x shifted 90° to the left — this equals y = cos x

## Reading Solutions from Graphs

To solve f(x) = g(x) graphically:
- Plot both curves
- x-coordinates of intersections are the solutions

> [!example] Solve x² − 3 = x graphically
> Draw y = x² − 3 and y = x on the same axes
> Read off x-values of intersections → these are the solutions

## Key Definitions

| Term | Definition |
|------|-----------|
| Parabola | Graph of a quadratic equation |
| Cubic | Polynomial of degree 3 |
| Hyperbola | Graph of y = k/x (two branches) |
| Asymptote | A line the graph approaches but never touches |
| Exponential | Function of the form aˣ |
| Amplitude | Half the height of a wave |
| Period | Distance after which a periodic function repeats |
| Transformation | A change in position, size, or orientation of a graph |
