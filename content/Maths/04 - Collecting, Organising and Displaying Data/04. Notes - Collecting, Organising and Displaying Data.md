## Types of Data

### Qualitative (Categorical) Data
Described in words — cannot be measured numerically.
- Examples: favourite colour, type of pet, gender, nationality

### Quantitative Data
Described with numbers.

| Sub-type | Description | Examples |
|---------|-------------|---------|
| **Discrete** | Can only take certain values (usually whole numbers — counted) | Number of siblings, shoe size, goals scored |
| **Continuous** | Can take any value in a range (measured) | Height, weight, time, temperature |

> [!tip] Key Distinction
> "How many?" → discrete. "How much?" or "How long/tall/heavy?" → continuous.

---

## Primary vs Secondary Data

| Type | Collected by | Examples |
|------|-------------|---------|
| **Primary** | The investigator themselves (first-hand) | Survey, experiment, observation |
| **Secondary** | Someone else, already exists | Internet, textbooks, government statistics |

---

## Sampling Methods

| Method | Description | Advantage |
|--------|-------------|-----------|
| **Random** | Every member has equal chance | Unbiased |
| **Systematic** | Every nth member of the list | Simple, practical |
| **Stratified** | Population divided into groups; sample proportional | Representative |
| **Convenience** | Whoever is available | Easy but biased |

> [!warning] Bias
> A sample is biased if it doesn't fairly represent the whole population. Small samples, non-random methods, and leading questions all introduce bias.

## Frequency Tables

A frequency table records how many times each value (or group) occurs.

> [!example] Ungrouped Frequency Table
> | Score | Tally | Frequency |
> |-------|-------|-----------|
> | 1 | II | 2 |
> | 2 | IIII | 4 |
> | 3 | III | 3 |
> | **Total** | | **9** |

### Grouped Frequency Table
Used for continuous data or many distinct values. Classes must be **non-overlapping** and **equal width** (usually).

> [!example]
> | Height (cm) | Frequency |
> |------------|-----------|
> | 150 ≤ h < 160 | 5 |
> | 160 ≤ h < 170 | 12 |
> | 170 ≤ h < 180 | 8 |

### Class Width and Midpoint
- Class width = upper boundary − lower boundary
- Midpoint = (lower boundary + upper boundary) / 2
- Use midpoints when estimating mean from grouped data (→ see Ch 12)

---

## Tally Charts
Group tallies in fives: `|||| |` = 5, making counting easier.

---

## Two-Way Tables
Organise data under two categories simultaneously.

> [!example]
> | | Pass | Fail | Total |
> |-|------|------|-------|
> | Male | 18 | 7 | 25 |
> | Female | 22 | 3 | 25 |
> | **Total** | 40 | 10 | **50** |

---

## Bar Charts

- Used for **discrete** or **categorical** data
- Bars have **equal width**, gaps between bars (for discrete), no gaps (for continuous → histogram)
- Height of bar = frequency (or relative frequency)
- **Comparative bar chart**: two sets of data side by side, same axes
- **Compound/stacked bar chart**: bars divided into parts

---

## Pie Charts

Each category is represented by a sector. The angle of each sector:

**Angle = (Frequency / Total) × 360°**

> [!example] 50 students, 20 chose Maths.
> Angle for Maths = (20/50) × 360° = **144°**

---

## Pictograms

- Use symbols/pictures to represent data
- A **key** must always be shown (e.g. ⭐ = 5 students)
- Half symbols can represent half the value

---

## Stem-and-Leaf Diagrams

Splits each data value into a **stem** (leading digit) and **leaf** (trailing digit).

> [!example] Data: 14, 17, 22, 25, 28, 31, 31, 36
>
> | Stem | Leaf |
> |------|------|
> | 1 | 4 7 |
> | 2 | 2 5 8 |
> | 3 | 1 1 6 |
>
> Key: 1|4 = 14

- **Back-to-back stem-and-leaf**: compare two datasets using the same stem, leaves either side
- Easily find median and mode from an ordered diagram

---

## Line Graphs

- Join data points with straight lines
- Best for showing **change over time** (time series)
- x-axis = time; y-axis = value

---

## Frequency Polygons

- Plot frequency against **midpoint** of each class
- Join plotted points with straight lines
- Used to compare frequency distributions visually

---

## Choosing the Right Chart

| Data type | Best chart |
|-----------|-----------|
| Categorical / discrete | Bar chart, pie chart, pictogram |
| Continuous, grouped | Histogram (Ch 20), frequency polygon |
| Change over time | Line graph |
| Comparing two groups | Back-to-back stem-and-leaf, comparative bar |
| Two variables, correlation | Scatter diagram (Ch 16) |

---

## Key Definitions

| Term | Definition |
|------|-----------|
| Population | The complete group being studied |
| Sample | A subset of the population |
| Census | Data collected from the entire population |
| Discrete data | Countable values with gaps between them |
| Continuous data | Measured values; any value in a range is possible |
| Frequency | How many times a value occurs |
| Class / interval | A range of values in a grouped frequency table |
| Midpoint | Middle value of a class interval |
| Tally | Marks used to count in groups of 5 |
