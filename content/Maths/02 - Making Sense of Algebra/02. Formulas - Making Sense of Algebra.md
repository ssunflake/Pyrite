## Index Laws

| Law | Rule |
|-----|------|
| Multiply | xᵐ × xⁿ = xᵐ⁺ⁿ |
| Divide | xᵐ ÷ xⁿ = xᵐ⁻ⁿ |
| Power of power | (xᵐ)ⁿ = xᵐⁿ |
| Zero index | x⁰ = 1 |
| Negative index | x⁻ⁿ = 1/xⁿ |
| Fractional index | x^(1/n) = ⁿ√x |
| Fractional index | x^(m/n) = (ⁿ√x)ᵐ |

## Expanding Brackets

```
Single:   a(b + c) = ab + ac

Double (FOIL):
(a + b)(c + d) = ac + ad + bc + bd
```

### Special Identities

```
(a + b)²     = a² + 2ab + b²
(a − b)²     = a² − 2ab + b²
(a + b)(a−b) = a² − b²
```

---

## Factorising Checklist

```
1. Common factor first   →  HCF(terms) × (...)
2. Difference of squares →  a² − b² = (a+b)(a−b)
3. Quadratic x²+bx+c    →  find p,q: pq=c, p+q=b  →  (x+p)(x+q)
4. Quadratic ax²+bx+c   →  split middle term using ac method
5. Grouping (4 terms)   →  group in pairs, factor each
```

---

## Changing the Subject — Strategy

```
1. Remove fractions (multiply through)
2. Expand brackets
3. Collect terms with subject on one side
4. Factor out the subject if it appears more than once
5. Divide to isolate subject
6. Square/root if needed
```

---

## Common Algebraic Errors to Avoid

| Wrong | Correct |
|-------|---------|
| (x+3)² = x²+9 | (x+3)² = x²+6x+9 |
| −x² = (−x)² | −x² = −(x²); (−x)² = x² |
| a(b−c) = ab−c | a(b−c) = ab − ac |
| √(a²+b²) = a+b | √(a²+b²) ≠ a+b (no simplification) |
