## Quadratic Formula

```
For ax² + bx + c = 0:

       −b ± √(b² − 4ac)
x  =  ─────────────────
              2a
```

## Discriminant (b² − 4ac)

```
> 0  →  two distinct real roots
= 0  →  one repeated root
< 0  →  no real roots
```

## Completing the Square

```
x² + bx + c  =  (x + b/2)² − (b/2)² + c
```

## Factorising Reference

```
Common factor:            HCF(terms) × (...)
Difference of squares:    a² − b²  =  (a+b)(a−b)
Perfect square:           a² + 2ab + b²  =  (a+b)²
                          a² − 2ab + b²  =  (a−b)²
Quadratic (monic):        x² + bx + c  =  (x+p)(x+q)   where pq=c, p+q=b
Quadratic (non-monic):    ac method — split middle term
Sum of cubes:             a³ + b³  =  (a+b)(a²−ab+b²)
Difference of cubes:      a³ − b³  =  (a−b)(a²+ab+b²)
```

## Simultaneous Equations — Steps

```
Elimination:
  1. Match coefficients (multiply equations if needed)
  2. Add or subtract to eliminate one variable
  3. Solve for remaining variable
  4. Substitute back

Substitution:
  1. Rearrange one equation to get y = f(x)
  2. Substitute into other equation
  3. Solve and back-substitute
```

## Linear Inequality Rules

```
Treat like equation EXCEPT:
  Multiply/divide by negative  →  FLIP the inequality sign
```

---

## Changing Subject Quick Checklist

```
1. Clear fractions (multiply through by denominator)
2. Expand all brackets
3. Move all terms with subject to one side
4. Factor out subject if it appears in multiple terms
5. Divide/root to isolate subject
```
