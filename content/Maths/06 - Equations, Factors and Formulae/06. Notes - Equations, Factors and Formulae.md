## Solving Linear Equations

The goal: isolate the variable. **Whatever you do to one side, do to the other.**

### One-step
> 3x = 12 → x = **4** (divide both sides by 3)
> x + 7 = 15 → x = **8** (subtract 7 from both sides)

### Two-step
> 2x + 5 = 11 → 2x = 6 → x = **3**

### With brackets
> 3(2x − 1) = 15
> 6x − 3 = 15
> 6x = 18 → x = **3**

### Variable on both sides
> 5x − 3 = 2x + 9
> Collect x terms: 3x = 12 → x = **4**

### Fractions
> (x + 2)/3 = 5
> x + 2 = 15 → x = **13**

> (2x − 1)/4 = (x + 3)/2
> Multiply through by 4: 2x − 1 = 2(x + 3) = 2x + 6
> Simplify: −1 = 6 → **no solution** (identity / contradiction check)

## Forming and Solving Equations from Word Problems

> [!example] A rectangle has length (2x + 3) cm and width (x − 1) cm. Perimeter = 28 cm. Find x.
> 2(2x + 3) + 2(x − 1) = 28
> 4x + 6 + 2x − 2 = 28
> 6x + 4 = 28
> 6x = 24 → **x = 4**

## Linear Inequalities

Solve like equations, but **flip the inequality sign when multiplying or dividing by a negative**.

> [!example] Solve: 3 − 2x > 7
> −2x > 4
> x < −2 ← **flip sign** (divided by −2)

### Number Line Representation
- Open circle ○ for strict inequality (< or >)
- Closed circle ● for inclusive inequality (≤ or ≥)
- Arrow in the direction of solutions

### Integer Solutions
List all integers satisfying an inequality.
> −3 < x ≤ 2: integers are **−2, −1, 0, 1, 2**

## Quadratic Equations

Standard form: **ax² + bx + c = 0**

### Method 1: Factorising
Write as (x + p)(x + q) = 0, then x = −p or x = −q.

> [!example] x² − 5x + 6 = 0
> (x − 2)(x − 3) = 0
> x = **2 or 3**

### Method 2: Quadratic Formula
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

> [!example] 2x² + 3x − 2 = 0
> a=2, b=3, c=−2
> discriminant = 3² − 4(2)(−2) = 9 + 16 = 25
> x = (−3 ± 5)/4
> x = 2/4 = **1/2** or x = −8/4 = **−2**

### Method 3: Completing the Square
x² + bx + c = 0 → (x + b/2)² − (b/2)² + c = 0

> [!example] x² + 6x + 7 = 0
> (x + 3)² − 9 + 7 = 0
> (x + 3)² = 2
> x + 3 = ±√2
> x = **−3 ± √2**

### The Discriminant (b² − 4ac)

| Discriminant | Nature of roots |
|-------------|----------------|
| > 0 | Two distinct real roots |
| = 0 | One repeated (equal) root |
| < 0 | No real roots |

## Simultaneous Equations (Linear)

Two equations, two unknowns — find values satisfying **both**.

### Method 1: Elimination
Add or subtract equations to eliminate one variable.

> [!example] 3x + 2y = 13 … (1)
> x − 2y = 3   … (2)
> Add: 4x = 16 → x = 4
> Sub into (2): 4 − 2y = 3 → y = 1/2

If coefficients don't match, multiply one (or both) equations first.

### Method 2: Substitution
Express one variable in terms of the other, substitute.

> [!example] y = 2x − 1 … (1)
> 3x + y = 14 … (2)
> Sub (1) into (2): 3x + (2x−1) = 14 → 5x = 15 → x = 3, y = 5

---

## Simultaneous Equations (Linear and Quadratic)

Substitute the linear equation into the quadratic.

> [!example] y = x + 3 and y = x²
> x² = x + 3 → x² − x − 3 = 0
> Use quadratic formula: x = (1 ± √13)/2

## Further Factorising Review

### Quadratic ax² + bx + c (ac method)
1. Multiply a × c
2. Find two numbers with product ac and sum b
3. Split middle term, group and factorise

> [!example] 6x² + 11x − 10
> a×c = −60; need product −60, sum +11 → 15 and −4
> = 6x² + 15x − 4x − 10
> = 3x(2x + 5) − 2(2x + 5)
> = **(3x − 2)(2x + 5)**

### Perfect Square and Sum/Difference Recognition

| Form | Factorisation |
|------|--------------|
| a² + 2ab + b² | (a + b)² |
| a² − 2ab + b² | (a − b)² |
| a² − b² | (a + b)(a − b) |
| a³ − b³ | (a − b)(a² + ab + b²) |
| a³ + b³ | (a + b)(a² − ab + b²) |

## Changing the Subject — Advanced

> [!example] Make x the subject of: p = √((x + q)/r)
> p² = (x + q)/r
> p²r = x + q
> x = **p²r − q**

> [!example] Make v the subject of: E = ½mv²
> 2E = mv²
> v² = 2E/m
> v = **√(2E/m)**

## Key Definitions

| Term | Definition |
|------|-----------|
| Linear equation | Contains no powers higher than 1 |
| Quadratic equation | Highest power is 2; standard form ax² + bx + c = 0 |
| Inequality | A statement comparing two expressions using <, >, ≤, ≥ |
| Simultaneous equations | Two or more equations with the same unknowns |
| Discriminant | b² − 4ac; determines nature of quadratic roots |
| Completing the square | Rewriting x² + bx as (x + b/2)² − (b/2)² |
| Elimination | Solving simultaneous equations by adding/subtracting to remove a variable |
| Substitution | Solving simultaneous equations by replacing one variable |
