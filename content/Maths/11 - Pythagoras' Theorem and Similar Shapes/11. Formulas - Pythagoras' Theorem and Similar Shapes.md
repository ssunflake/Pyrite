## Pythagoras' Theorem

```
In a right triangle with hypotenuse c:
  a² + b²  =  c²
  c  =  √(a² + b²)
  a  =  √(c² − b²)
```

## Distances

```
2D distance:     d  =  √[(x₂−x₁)² + (y₂−y₁)²]
Cuboid diagonal: d  =  √(l² + w² + h²)
Rectangle diagonal: d = √(l² + w²)
```

## Pythagorean Triples

```
(3,4,5)   (5,12,13)   (8,15,17)   (7,24,25)
and all integer multiples
```

## Similar Shapes

```
Linear scale factor:  k  =  corresponding length ratio

Lengths / perimeters  →  × k
Areas                 →  × k²
Volumes               →  × k³

Finding k from areas:   k  =  √(area ratio)
Finding k from volumes: k  =  ∛(volume ratio)
```

## Similar Triangle Tests

```
AA   →  2 pairs equal angles (automatically similar)
SSS  →  all 3 side ratios equal
SAS  →  2 sides in ratio + included angle equal
```

## Parallel Line in Triangle

```
If PQ ∥ BC in △ABC, then:
AP/AB = AQ/AC = PQ/BC
```
