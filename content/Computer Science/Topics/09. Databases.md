## 9.1 Database Structure

> A **database** is an organised collection of structured data, stored electronically and managed by a **Database Management System (DBMS)**.

### Flat-file vs Relational Databases

| | Flat-file | Relational |
|--|-----------|-----------|
| Tables | One table | Multiple linked tables |
| Data redundancy | High (data repeated) | Low (data stored once) |
| Complexity | Simple | More complex |
| Use case | Simple data, small scale | Complex applications |

### Key Terms

| Term | Definition |
|------|-----------|
| **Table (entity)** | A structured set of data in rows and columns |
| **Record (row)** | One complete set of data about one item/person |
| **Field (column/attribute)** | One piece of information in each record |
| **Primary key** | A field that uniquely identifies each record in a table |
| **Foreign key** | A field in one table that links to the primary key of another |

### Example Table: Students

| StudentID | FirstName | LastName | DateOfBirth | Class |
|-----------|-----------|----------|-------------|-------|
| 1001 | Aiden | Smith | 2009-03-15 | 10A |
| 1002 | Priya | Sharma | 2008-11-22 | 10B |
| 1003 | James | Wong | 2009-07-04 | 10A |

- **StudentID** = Primary key (unique for each student)
- Each row = one record
- Each column = one field

### Relational Databases

Two tables linked by a foreign key:

**Students table** (StudentID, Name, ClassID ‚Üź foreign key)
**Classes table** (ClassID ‚Üź primary key, ClassName, TeacherName)

## 9.2 SQL

> **SQL** (Structured Query Language) is the standard language for querying and manipulating relational databases.

SQL is **not case-sensitive** for keywords, but table/field names may be.

---

## 9.3 SELECT ‚Ä¶ FROM

Retrieves fields from a table.

```sql
SELECT * FROM Students
```
Returns ALL fields for ALL records in Students.

```sql
SELECT FirstName, LastName FROM Students
```
Returns only the FirstName and LastName fields.

---

## 9.4 SELECT ‚Ä¶ FROM ‚Ä¶ WHERE

Filters records that meet a condition.

```sql
SELECT * FROM Students
WHERE Class = '10A'
```

```sql
SELECT FirstName, LastName FROM Students
WHERE DateOfBirth > '2009-01-01'
```

### Multiple Conditions

```sql
SELECT * FROM Students
WHERE Class = '10A' AND DateOfBirth > '2009-01-01'
```

```sql
SELECT FirstName FROM Students
WHERE Class = '10A' OR Class = '10B'
```

### Comparison Operators in SQL

| Operator | Meaning |
|----------|---------|
| `=` | Equal to |
| `<>` or `!=` | Not equal to |
| `<` | Less than |
| `>` | Greater than |
| `<=` | Less than or equal to |
| `>=` | Greater than or equal to |
| `LIKE` | Pattern match (% = wildcard) |

```sql
-- Students whose last name starts with 'S'
SELECT * FROM Students
WHERE LastName LIKE 'S%'
```

---

## 9.5 ORDER BY

Sorts results.

```sql
-- Sort by last name A-Z (ascending)
SELECT * FROM Students
ORDER BY LastName ASC

-- Sort by date of birth newest first (descending)
SELECT * FROM Students
ORDER BY DateOfBirth DESC
```

- `ASC` = ascending (A‚ÜíZ, 0‚Üí9) ‚ÄĒ **default**
- `DESC` = descending (Z‚ÜíA, 9‚Üí0)

---

## 9.6 SUM

Calculates the total of a numeric field.

```sql
-- Total of all scores in Results table
SELECT SUM(Score) FROM Results

-- Total score for one student
SELECT SUM(Score) FROM Results
WHERE StudentID = 1001
```

---

## 9.7 COUNT

Counts the number of records.

```sql
-- Count all students
SELECT COUNT(*) FROM Students

-- Count students in class 10A
SELECT COUNT(*) FROM Students
WHERE Class = '10A'
```

### Other Aggregate Functions

| Function | Purpose |
|----------|---------|
| `SUM(field)` | Total of values |
| `COUNT(*)` | Number of records |
| `AVG(field)` | Average of values |
| `MAX(field)` | Largest value |
| `MIN(field)` | Smallest value |

```sql
-- Average, max and min score
SELECT AVG(Score), MAX(Score), MIN(Score)
FROM Results
WHERE StudentID = 1001
```

## Putting It Together ‚ÄĒ Complex Queries

```sql
-- Names of students in class 10A who scored above 70,
-- ordered by score descending

SELECT Students.FirstName, Students.LastName, Results.Score
FROM Students, Results
WHERE Students.StudentID = Results.StudentID
AND Students.Class = '10A'
AND Results.Score > 70
ORDER BY Results.Score DESC
```
