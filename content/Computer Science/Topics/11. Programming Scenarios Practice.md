## 11.2 Method 1: Identifying Inputs, Processes and Outputs

**Scenario: Number Guessing Game**
> The computer stores a number 1–100. The user has 10 attempts. After each guess, tell them if they're too high, too low or correct. If they fail in 10, reveal the answer.

| | Detail |
|--|--------|
| **Inputs** | User's guess (integer 1–100) |
| **Processes** | Compare guess to stored number; track attempts; validate range |
| **Outputs** | "Too high" / "Too low" / "Correct in X attempts" / "Answer was X" |

**Pseudocode:**
```
StoredNumber <- RANDOM(1, 100)
Attempts <- 0
Correct <- FALSE

WHILE Attempts < 10 AND Correct = FALSE DO
    OUTPUT("Enter your guess (1-100)")
    INPUT Guess
    Attempts <- Attempts + 1

    IF Guess = StoredNumber THEN
        OUTPUT("Correct! You got it in " & Attempts & " attempts")
        Correct <- TRUE
    ELSEIF Guess < StoredNumber THEN
        OUTPUT("Too low")
    ELSE
        OUTPUT("Too high")
    ENDIF
ENDWHILE

IF Correct = FALSE THEN
    OUTPUT("Out of attempts! The answer was " & StoredNumber)
ENDIF
```

```python
# Python
import random

storedNumber = random.randint(1, 100)
attempts = 0
correct = False

while attempts < 10 and not correct:
    guess = int(input("Enter your guess (1-100): "))
    attempts += 1

    if guess == storedNumber:
        print("Correct! You got it in", attempts, "attempts")
        correct = True
    elif guess < storedNumber:
        print("Too low")
    else:
        print("Too high")

if not correct:
    print("Out of attempts! The answer was", storedNumber)
```

## 11.3 Writing the Code

### Full Worked Scenario: Student Grade Calculator

**Scenario:** Ask for 5 student test scores (0–100). Output the total, average, highest and lowest score, and a grade (A=90+, B=70+, C=50+, F=below 50) for the average.

```python
# Python - complete grade calculator
scores = []
total = 0
highest = -1
lowest = 999

# Input validation and collection
for i in range(5):
    while True:
        score = int(input(f"Enter score {i+1} (0-100): "))
        if 0 <= score <= 100:
            break
        print("Invalid! Score must be 0-100")
    scores.append(score)
    total += score
    if score > highest:
        highest = score
    if score < lowest:
        lowest = score

# Calculate average
average = total / 5

# Determine grade
if average >= 90:
    grade = "A"
elif average >= 70:
    grade = "B"
elif average >= 50:
    grade = "C"
else:
    grade = "F"

# Output results
print("--- Results ---")
print("Total:", total)
print("Average:", round(average, 1))
print("Highest:", highest)
print("Lowest:", lowest)
print("Grade:", grade)
```

```java
// Java - grade calculator
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
int[] scores = new int[5];
int total = 0, highest = -1, lowest = 999;

for (int i = 0; i < 5; i++) {
    int score;
    do {
        System.out.println("Enter score " + (i+1) + " (0-100):");
        score = Integer.parseInt(sc.nextLine());
        if (score < 0 || score > 100)
            System.out.println("Invalid!");
    } while (score < 0 || score > 100);

    scores[i] = score;
    total += score;
    if (score > highest) highest = score;
    if (score < lowest) lowest = score;
}

double average = total / 5.0;
String grade;
if (average >= 90) grade = "A";
else if (average >= 70) grade = "B";
else if (average >= 50) grade = "C";
else grade = "F";

System.out.println("Total: " + total);
System.out.println("Average: " + Math.round(average * 10.0) / 10.0);
System.out.println("Highest: " + highest);
System.out.println("Lowest: " + lowest);
System.out.println("Grade: " + grade);
```

### Scenario: Login System with File Storage

```python
# Python - login system saving credentials to file
import os

FILENAME = "users.txt"

def registerUser():
    username = input("Choose a username: ")
    password = input("Choose a password (min 8 chars): ")
    while len(password) < 8:
        print("Password too short!")
        password = input("Try again: ")
    with open(FILENAME, "a") as f:
        f.write(username + "," + password + "\n")
    print("Registered successfully!")

def loginUser():
    username = input("Username: ")
    password = input("Password: ")
    if not os.path.exists(FILENAME):
        print("No users registered yet.")
        return
    with open(FILENAME, "r") as f:
        for line in f:
            stored = line.strip().split(",")
            if stored[0] == username and stored[1] == password:
                print("Login successful! Welcome,", username)
                return
    print("Invalid username or password.")

# Main menu
print("1. Register  2. Login")
choice = input("Choose: ")
if choice == "1":
    registerUser()
elif choice == "2":
    loginUser()
else:
    print("Invalid choice")
```

## Exam Technique: Tackling Programming Questions

### Step-by-step approach:

1. **Read the scenario carefully** — underline inputs, processes and outputs
2. **Identify validation needed** — range checks, length checks, type checks
3. **Choose your loop type:**
   - Count-controlled (FOR) when you know how many times
   - Pre-condition (WHILE) when you check before
   - Post-condition (REPEAT/DO-WHILE) when you must run at least once
4. **Write pseudocode first** if the question asks for it — focus on logic, not syntax
5. **Test with boundary and abnormal data** in your trace table

### Common Patterns to Know by Heart

| Pattern | Code structure |
|---------|---------------|
| Input validation | WHILE/DO-WHILE with condition check |
| Count items | `count = 0` then `count += 1` inside loop |
| Sum items | `total = 0` then `total += value` inside loop |
| Find max | `largest = -1` then `IF value > largest: largest = value` |
| Find min | `smallest = 999` then `IF value < smallest: smallest = value` |
| Linear search | FOR loop with IF; set found flag |
| Bubble sort | Nested FOR with swap and change flag |
| Menu system | WHILE loop with CASE/if-elif for options |

## Key Definitions

| Term | Definition |
|------|-----------|
| **Validation** | Checking data is reasonable and within acceptable bounds before processing |
| **Verification** | Checking data has been entered accurately (e.g. double entry, visual check) |
| **Range check** | Validates data is within a minimum and maximum value |
| **Length check** | Validates string is the right length |
| **Type check** | Validates data is the correct data type |
| **Presence check** | Validates that a field is not empty |
| **Format check** | Validates data matches a required pattern (e.g. date DD/MM/YYYY) |
| **Check digit** | Extra digit calculated from others to detect input errors (e.g. barcodes) |
