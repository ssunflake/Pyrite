## 6.1 Automated Systems

> An **automated system** is a system that performs tasks or makes decisions with minimal or no human intervention, using a combination of sensors, processors, and actuators.

### Components of an Automated System

| Component | Role | Examples |
|-----------|------|---------|
| **Sensor** | Collects data from the environment (input) | Temperature, motion, light, pressure |
| **Processor/Microcontroller** | Processes sensor data and makes decisions | Arduino, Raspberry Pi, embedded chip |
| **Actuator** | Takes physical action in response to processor output | Motor, pump, valve, alarm, light |

### Feedback Loop

```
Sensor → Processor → Actuator → Environment → back to Sensor
```

Most automated systems use **closed-loop control** (feedback):
- System continuously monitors its own output and adjusts

**Example: Central Heating**
1. Temperature sensor reads room temperature
2. Processor compares to desired (set) temperature
3. If too cold → actuator (boiler) turns on
4. If warm enough → boiler turns off
5. Cycle repeats

### Examples of Automated Systems

| System | Sensors | Actuators |
|--------|---------|----------|
| **Automatic irrigation** | Soil moisture, temperature | Water pumps, valves |
| **Traffic lights** | Pressure sensors in road, cameras | Traffic light signals |
| **Autopilot (aircraft)** | GPS, altitude, speed, direction | Ailerons, elevators, throttle |
| **Manufacturing robots** | Position, force, vision | Arms, welders, drill heads |
| **Smart home** | Motion, light, temperature | Lights, HVAC, locks |

### Advantages and Disadvantages of Automation

| Advantages | Disadvantages |
|-----------|--------------|
| Works continuously (24/7) without fatigue | High initial setup cost |
| Consistent, repeatable precision | Can malfunction if sensor fails |
| Reduces labour costs long-term | Job displacement for workers |
| Safer in hazardous environments | Requires skilled programmers to maintain |
| Can respond faster than humans | No human judgement |

## 6.2 Robotics

> A **robot** is a machine that can be programmed to carry out complex tasks automatically, often able to sense and interact with its environment.

### Components of a Robot
| Component | Function |
|-----------|---------|
| **Sensors** | Gather data (proximity, vision, force, GPS) |
| **Effectors/actuators** | Physical movement (arms, wheels, grippers) |
| **Controller** | Processes sensor data and runs the program |
| **Power source** | Battery, electric motor, pneumatic |

### Types of Robots

| Type | Description | Examples |
|------|-------------|---------|
| **Industrial** | Fixed, high-precision automation | Car assembly, welding, painting |
| **Mobile** | Can move around environment | Warehouse robots (Amazon Kiva) |
| **Autonomous** | Makes decisions without human control | Self-driving cars, drones |
| **Collaborative (cobots)** | Works alongside humans | Surgical assist robots |

## 6.3 Robots in Context

### Advantages of Robots

| Advantage | Detail |
|-----------|--------|
| **Precision** | Repeatable accuracy beyond human capability |
| **Hazardous environments** | Nuclear plants, bomb disposal, deep sea, space |
| **Consistent quality** | No fatigue, no human error |
| **Speed** | Can operate continuously at high speed |
| **Cost (long-term)** | Replaces expensive human labour over time |

### Disadvantages of Robots

| Disadvantage | Detail |
|-------------|--------|
| **High upfront cost** | Hardware, programming, installation |
| **Job displacement** | Reduces need for human workers |
| **Lack of judgement** | Cannot adapt to unexpected situations as well as humans |
| **Maintenance** | Breakdowns require specialist technical repair |
| **Programming required** | Complex behaviour requires significant development |

### Ethical Considerations
- Large-scale automation → unemployment in manufacturing sectors
- Autonomous robots (e.g. military drones) raise questions about responsibility
- Data privacy in robots with cameras/microphones in homes

## 6.4 Artificial Intelligence

> **Artificial Intelligence (AI)** is the simulation of human intelligence processes by machines, especially computer systems.

### What AI can do
- **Machine learning:** Systems that learn from data without being explicitly programmed
- **Natural language processing:** Understanding and generating human language (e.g. chatbots)
- **Computer vision:** Interpreting visual input (facial recognition, medical imaging)
- **Expert systems:** Making decisions within a specific domain (medical diagnosis, chess)
- **Pattern recognition:** Spam filters, fraud detection, recommendation systems

### Machine Learning Overview
- Training data → model learns patterns → model makes predictions on new data
- **Supervised learning:** Labelled training examples (cat/dog classifier)
- **Unsupervised learning:** Finds patterns without labels (clustering)

### Expert Systems

| Component | Role |
|-----------|------|
| **Knowledge base** | Facts and rules about the domain |
| **Inference engine** | Applies rules to reach conclusions |
| **User interface** | How user asks questions and gets answers |

**Examples:** Medical diagnosis systems, legal advice tools

### AI Applications

| Application | AI Type |
|-------------|---------|
| Spam filters | Machine learning |
| Facial recognition | Computer vision |
| Voice assistants (Siri, Alexa) | NLP + ML |
| Recommendation engines (Netflix, Spotify) | ML |
| Self-driving vehicles | Computer vision + ML |
| Chess/game playing (DeepMind AlphaGo) | Reinforcement learning |
| Fraud detection | ML + pattern recognition |

### Benefits of AI

| Benefit | Example |
|---------|---------|
| Performs tasks humans can't do quickly | Analysing billions of data points |
| Operates in dangerous environments | Bomb disposal, surgery assistance |
| Available 24/7 | Customer service chatbots |
| Learns and improves over time | Spam filters get better |

### Concerns about AI

| Concern | Detail |
|---------|--------|
| **Bias** | AI trained on biased data perpetuates discrimination |
| **Job displacement** | White-collar jobs (writing, legal) increasingly at risk |
| **Transparency** | "Black box" models — unexplainable decisions |
| **Privacy** | Facial recognition, surveillance without consent |
| **Dependence** | Over-reliance on AI systems that can fail |
| **Autonomous weapons** | AI in military systems raises ethical questions |

## Key Definitions

| Term | Definition |
|------|-----------|
| **Automated system** | System that performs tasks with minimal human intervention |
| **Sensor** | Device that detects and measures real-world data |
| **Actuator** | Device that takes physical action in response to a control signal |
| **Robot** | Programmable machine that can carry out physical tasks, often with sensing capability |
| **Artificial intelligence** | Simulation of human intelligent behaviour by computer systems |
| **Machine learning** | AI technique where systems learn patterns from data |
| **Expert system** | AI system that mimics decision-making of a human expert in a specific domain |
