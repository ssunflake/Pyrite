## 8.1 Programming Concepts — Variables and Constants

> A **variable** is a named memory location that can store data which can change during the program.
> A **constant** is a named memory location storing data that **cannot** change during the program.

```
# Pseudocode
Number <- 10
Colour <- "red"
Price <- 22.2
CONSTANT MaxValue <- 100
```

```python
# Python
number = 10
colour = "red"
price = 22.2
# Python has no built-in constant; use UPPER_CASE as convention
MAX_VALUE = 100
```

```vb
' VB.NET
Dim number As Integer = 10
Dim colour As String = "red"
Dim price As Single = 22.2
Const MAX_VALUE As Integer = 100
```

```java
// Java
Integer number = 10;
String colour = "red";
Double price = 22.2;
final Integer MAX_VALUE = 100;
```

## 8.4 Arithmetic Operators

| Operator | Meaning | Example | Result |
|----------|---------|---------|--------|
| `+` | Addition | 10 + 3 | 13 |
| `-` | Subtraction | 10 - 3 | 7 |
| `*` | Multiplication | 10 * 3 | 30 |
| `/` | Division | 10 / 3 | 3.33 |
| `DIV` | Integer division (ignore remainder) | DIV(11, 3) | 3 |
| `MOD` | Remainder after division | MOD(11, 3) | 2 |

**MOD tip:** `MOD(n, 2) = 0` means n is **even**; `MOD(n, 2) = 1` means n is **odd**.



```java
// Java
```

## 8.6 Selection

### IF / ELSE / ELSEIF

```
# Pseudocode
IF Age < 18 THEN
    OUTPUT("You are a minor")
ELSEIF Age < 65 THEN
    OUTPUT("You are an adult")
ELSE
    OUTPUT("You are a senior")
ENDIF
```

```python
# Python
age = int(input("Enter age: "))
if age < 18:
    print("You are a minor")
elif age < 65:
    print("You are an adult")
else:
    print("You are a senior")
```

```vb
' VB.NET
Dim age As Integer = Console.ReadLine()
If age < 18 Then
    Console.WriteLine("You are a minor")
ElseIf age < 65 Then
    Console.WriteLine("You are an adult")
Else
    Console.WriteLine("You are a senior")
End If
```

```java
// Java
Scanner sc = new Scanner(System.in);
int age = Integer.parseInt(sc.nextLine());
if (age < 18) {
    System.out.println("You are a minor");
} else if (age < 65) {
    System.out.println("You are an adult");
} else {
    System.out.println("You are a senior");
}
```

### CASE / SELECT CASE / Switch

```
# Pseudocode
CASE OF Day:
    1: OUTPUT("Monday")
    2: OUTPUT("Tuesday")
    3: OUTPUT("Wednesday")
    OTHERWISE OUTPUT("Invalid")
ENDCASE
```

```python
# Python (no CASE — use elif chain or match in Python 3.10+)
day = int(input())
if day == 1:
    print("Monday")
elif day == 2:
    print("Tuesday")
elif day == 3:
    print("Wednesday")
else:
    print("Invalid")
```

```vb
' VB.NET
Dim day As Integer = Console.ReadLine()
Select Case day
    Case 1
        Console.WriteLine("Monday")
    Case 2
        Console.WriteLine("Tuesday")
    Case 3
        Console.WriteLine("Wednesday")
    Case Else
        Console.WriteLine("Invalid")
End Select
```

```java
// Java
int day = Integer.parseInt(sc.nextLine());
switch (day) {
    case 1: System.out.println("Monday"); break;
    case 2: System.out.println("Tuesday"); break;
    case 3: System.out.println("Wednesday"); break;
    default: System.out.println("Invalid");
}
```

## 8.7 Iteration (Loops)

### FOR Loop (count-controlled)

```
# Pseudocode
FOR Count <- 1 TO 10
    OUTPUT Count
NEXT Count
```

```python
# Python
for count in range(1, 11):
    print(count)
```

```vb
' VB.NET
For count = 1 To 10
    Console.WriteLine(count)
Next
```

```java
// Java
for (int count = 1; count <= 10; count++) {
    System.out.println(count);
}
```

### WHILE Loop (pre-condition — tests BEFORE running)

```
# Pseudocode
WHILE Number > 0 DO
    OUTPUT Number
    Number <- Number - 1
ENDWHILE
```

```python
# Python
while number > 0:
    print(number)
    number = number - 1
```

```vb
' VB.NET
While number > 0
    Console.WriteLine(number)
    number = number - 1
End While
```

```java
// Java
while (number > 0) {
    System.out.println(number);
    number--;
}
```

### REPEAT…UNTIL Loop (post-condition — tests AFTER running, always runs at least once)

```
# Pseudocode
REPEAT
    OUTPUT("Enter a number > 0")
    INPUT Number
UNTIL Number > 0
```

```python
# Python (simulate with while True + break)
while True:
    number = int(input("Enter a number > 0: "))
    if number > 0:
        break
```

```vb
' VB.NET
Dim number As Integer
Do
    Console.WriteLine("Enter a number > 0")
    number = Console.ReadLine()
Loop Until number > 0
```

```java
// Java
int number;
do {
    System.out.println("Enter a number > 0");
    number = Integer.parseInt(sc.nextLine());
} while (number <= 0);
```

## 8.8–8.9 Totalling and Counting

```python
# Python - totalling and counting even numbers from user input
total = 0
count = 0
for i in range(10):
    value = int(input("Enter number: "))
    total = total + value        # totalling
    if value % 2 == 0:
        count = count + 1        # counting evens
print("Total:", total)
print("Even numbers:", count)
```

```vb
' VB.NET
Dim total As Integer = 0
Dim count As Integer = 0
For i = 1 To 10
    Dim value As Integer = Console.ReadLine()
    total = total + value
    If value Mod 2 = 0 Then count = count + 1
Next
Console.WriteLine("Total: " & total)
Console.WriteLine("Even count: " & count)
```

```java
// Java
int total = 0, count = 0;
Scanner sc = new Scanner(System.in);
for (int i = 0; i < 10; i++) {
    int value = Integer.parseInt(sc.nextLine());
    total += value;
    if (value % 2 == 0) count++;
}
System.out.println("Total: " + total);
System.out.println("Even count: " + count);
```

## 8.10 String Manipulation

| Operation | Pseudocode | Python | VB.NET | Java |
|-----------|------------|--------|--------|------|
| Length | `LENGTH(str)` | `len(str)` | `Len(str)` | `str.length()` |
| Substring | `SUBSTRING(str, start, len)` | `str[start:end]` | `Mid(str, start, len)` | `str.substring(start, end)` |
| Uppercase | `UCASE(str)` | `str.upper()` | `UCase(str)` | `str.toUpperCase()` |
| Lowercase | `LCASE(str)` | `str.lower()` | `LCase(str)` | `str.toLowerCase()` |

```python
# Python string examples
name = "Hello World"
print(len(name))          # 11
print(name[0:5])          # Hello
print(name.upper())       # HELLO WORLD
print(name.lower())       # hello world
```

```vb
' VB.NET
Dim name As String = "Hello World"
Console.WriteLine(Len(name))          ' 11
Console.WriteLine(Mid(name, 1, 5))    ' Hello (1-indexed!)
Console.WriteLine(UCase(name))        ' HELLO WORLD
```

```java
// Java
String name = "Hello World";
System.out.println(name.length());           // 11
System.out.println(name.substring(0, 5));    // Hello
System.out.println(name.toUpperCase());      // HELLO WORLD
```

## 8.12 Subroutines (Procedures and Functions)

> A **subroutine** is a named block of code that can be called from anywhere in the program.
> - **Procedure** = does NOT return a value
> - **Function** = DOES return a value

### Procedure (no return value)

```python
# Python
def greet(name):
    print("Hello,", name)

greet("Sunflake")
```

```vb
' VB.NET
Sub greet(name As String)
    Console.WriteLine("Hello, " & name)
End Sub

greet("Sunflake")
```

```java
// Java
public static void greet(String name) {
    System.out.println("Hello, " + name);
}
// In main:
greet("Sunflake");
```

### Function (returns a value)

```python
# Python
def multiply(a, b):
    return a * b

result = multiply(5, 3)
print(result)  # 15
```

```vb
' VB.NET
Function multiply(a As Integer, b As Integer) As Integer
    Return a * b
End Function

Dim result As Integer = multiply(5, 3)
Console.WriteLine(result)
```

```java
// Java
public static int multiply(int a, int b) {
    return a * b;
}
// In main:
int result = multiply(5, 3);
System.out.println(result);
```

### Scope: Global vs Local

- **Global:** declared outside functions; accessible everywhere
- **Local:** declared inside a function; only accessible within it

```python
# Python scope
globalVar = "I'm global"

def showScope():
    localVar = "I'm local"
    print(globalVar)   # Works
    print(localVar)    # Works

showScope()
# print(localVar)  # Would cause NameError!
```

## 8.14 Maintainable Programs

Good programs should be easy to read and update later:

| Technique | How |
|-----------|-----|
| **Meaningful identifiers** | `totalScore` not `x`; `calculateAverage()` not `func1()` |
| **Comments** | Explain complex sections; not every line needed |
| **Subroutines** | Split code into logical blocks; reduces repetition |
| **Indentation** | Visually shows structure (mandatory in Python!) |

```python
# Python - well commented, meaningful names
MAX_ATTEMPTS = 3  # constant - maximum allowed login attempts

def checkPassword(inputPassword, storedPassword):
    """Returns True if passwords match, False otherwise."""
    return inputPassword == storedPassword

# Main program
attempts = 0
while attempts < MAX_ATTEMPTS:
    password = input("Enter password: ")
    if checkPassword(password, "secret123"):
        print("Access granted")
        break
    attempts += 1
    print("Incorrect. Attempts left:", MAX_ATTEMPTS - attempts)
```

## 8.15 Arrays

> An **array** is a data structure that stores multiple values of the **same data type** under one identifier.

### 1D Arrays

```python
# Python - 1D array (list)
colours = ["red", "blue", "green", "yellow"]
print(colours[0])        # red (0-indexed)
colours[2] = "purple"    # change element

# Loop through array
for i in range(len(colours)):
    print(colours[i])
```

```vb
' VB.NET - 1D array
Dim colours(3) As String   ' 4 elements: index 0 to 3
colours(0) = "red"
colours(1) = "blue"
colours(2) = "green"
colours(3) = "yellow"

For i = 0 To 3
    Console.WriteLine(colours(i))
Next
```

```java
// Java - 1D array
String[] colours = new String[4];
colours[0] = "red";
colours[1] = "blue";
colours[2] = "green";
colours[3] = "yellow";

for (int i = 0; i < colours.length; i++) {
    System.out.println(colours[i]);
}
```

### 2D Arrays

Think of a 2D array as a **table with rows and columns**.
Access: `array[row][column]`

```python
# Python - 2D array
grid = [[0] * 3 for _ in range(3)]  # 3x3 grid of zeros
grid[0][0] = 10
grid[1][2] = 5

# Loop through all elements
for row in range(3):
    for col in range(3):
        print(grid[row][col], end=" ")
    print()
```

```vb
' VB.NET - 2D array
Dim grid(2, 2) As Integer   ' 3x3
grid(0, 0) = 10
grid(1, 2) = 5

For row = 0 To 2
    For col = 0 To 2
        Console.Write(grid(row, col) & " ")
    Next
    Console.WriteLine()
Next
```

```java
// Java - 2D array
int[][] grid = new int[3][3];
grid[0][0] = 10;
grid[1][2] = 5;

for (int row = 0; row < 3; row++) {
    for (int col = 0; col < 3; col++) {
        System.out.print(grid[row][col] + " ");
    }
    System.out.println();
}
```

## 8.16 File Handling

### Reading from a File

```
# Pseudocode
OPEN "data.txt"
Data <- READ("data.txt")
CLOSE "data.txt"
OUTPUT(Data)
```

```python
# Python - read from file
theFile = open("data.txt", "r")
data = theFile.read()
theFile.close()
print(data)
```

```vb
' VB.NET - read from file
Dim theFile As New System.IO.StreamReader("data.txt")
Dim data As String = theFile.ReadLine()
theFile.Close()
Console.WriteLine(data)
```

```java
// Java - read from file
try {
    FileReader f = new FileReader("data.txt");
    BufferedReader reader = new BufferedReader(f);
    String data = reader.readLine();
    reader.close();
    System.out.println(data);
} catch (Exception e) {
    System.err.println("File not found");
}
```

### Writing to a File

```python
# Python - write to file
theFile = open("data.txt", "w")
theFile.write("Hello, World!")
theFile.close()
```

```vb
' VB.NET - write to file
Dim fileWriter As New System.IO.StreamWriter("data.txt")

```

```java
// Java - write to file
try {
```

## Key Definitions

| Term | Definition |
|------|-----------|
| **Variable** | Named memory location whose value can change during program execution |
| **Constant** | Named memory location whose value cannot change |
| **Data type** | Classification of data (integer, string, Boolean, etc.) |
| **Sequence** | Programming construct where instructions run once in order |
| **Selection** | Programming construct choosing which code to run based on a condition |
| **Iteration** | Programming construct repeating code (loops) |
| **Subroutine** | Named block of code that can be called from elsewhere |
| **Procedure** | Subroutine that does not return a value |
| **Function** | Subroutine that returns a value |
| **Parameter** | Value passed into a subroutine when called |
| **Array** | Data structure storing multiple values of the same type under one identifier |
| **Global scope** | Variable accessible from any part of the program |
| **Local scope** | Variable only accessible within the subroutine where it is declared |
