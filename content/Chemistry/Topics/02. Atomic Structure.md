## 2.1 Atoms and Elements

> An **atom** is the smallest particle of an element that retains the chemical properties of that element.

### Subatomic Particles

| Particle | Symbol | Relative mass | Relative charge | Location |
|----------|--------|--------------|-----------------|---------|
| **Proton** | p | 1 | +1 | Nucleus |
| **Neutron** | n | 1 | 0 | Nucleus |
| **Electron** | e | 1/1836 ≈ 0 | −1 | Shells around nucleus |

### Key Atomic Numbers

| Term | Symbol | Definition |
|------|--------|-----------|
| **Atomic number** (proton number) | Z | Number of protons in an atom |
| **Mass number** (nucleon number) | A | Total number of protons + neutrons |
| **Number of neutrons** | N | A − Z |
| **Number of electrons** | = Z | (in a neutral atom) |

### Nuclide Notation
$$^A_Z X$$

**Example:** $^{23}_{11}\text{Na}$
- Atomic number = 11 → 11 protons, 11 electrons
- Mass number = 23 → 23 − 11 = 12 neutrons

### Elements and Compounds

| Term | Definition | Example |
|------|-----------|---------|
| **Element** | Substance containing only one type of atom | Iron (Fe), Oxygen (O₂) |
| **Compound** | Substance containing two or more elements chemically bonded | Water (H₂O), NaCl |
| **Molecule** | Group of atoms bonded covalently | O₂, H₂O, CH₄ |

## 2.2 Isotopes

> **Isotopes** are atoms of the **same element** that have the **same atomic number** but **different mass numbers** (different numbers of neutrons).

### Examples of Isotopes

| Element | Isotope 1 | Isotope 2 | Isotope 3 |
|---------|----------|----------|----------|
| Hydrogen | ¹H (protium) | ²H (deuterium) | ³H (tritium) |
| Carbon | ¹²C | ¹³C | ¹⁴C (radioactive) |
| Chlorine | ³⁵Cl | ³⁷Cl | — |
| Uranium | ²³⁵U | ²³⁸U | — |

### Properties of Isotopes

**Same:** Chemical properties (same electron configuration → same reactions), atomic number, number of electrons

**Different:** Mass number, number of neutrons, physical properties (density, rate of diffusion), nuclear stability

### Relative Atomic Mass (Aᵣ)

> The **relative atomic mass** is the **weighted average** mass of all naturally occurring isotopes of an element, relative to 1/12 the mass of a carbon-12 atom.

$$A_r = \frac{\sum (\text{isotope mass} \times \text{percentage abundance})}{100}$$

**Example — Chlorine:**
- ³⁵Cl: 75% abundance, ³⁷Cl: 25% abundance
$$A_r(\text{Cl}) = \frac{(35 \times 75) + (37 \times 25)}{100} = \frac{2625 + 925}{100} = 35.5$$

### Uses of Isotopes
| Isotope | Use |
|---------|-----|
| ¹⁴C | Carbon dating (archaeology) |
| ²³⁵U | Nuclear fuel (fission) |
| ¹³¹I | Medical: treating thyroid cancer |
| ⁶⁰Co | Medical: radiotherapy for cancer |
| ²H | NMR spectroscopy tracer |

## 2.3 Electronic Configuration of Elements

> **Electronic configuration** describes how electrons are arranged in energy levels (shells) around the nucleus.

### Shell Rules
| Shell | Maximum electrons |
|-------|------------------|
| Shell 1 (closest to nucleus) | 2 |
| Shell 2 | 8 |
| Shell 3 | 8 (for IGCSE purposes) |
| Shell 4 | 2 (for first 20 elements) |

### Electronic Configurations (First 20 Elements)

| Element | Z | Configuration | Dot-cross notation |
|---------|---|--------------|-------------------|
| H | 1 | 1 | |
| He | 2 | 2 | |
| Li | 3 | 2,1 | |
| Be | 4 | 2,2 | |
| B | 5 | 2,3 | |
| C | 6 | 2,4 | |
| N | 7 | 2,5 | |
| O | 8 | 2,6 | |
| F | 9 | 2,7 | |
| Ne | 10 | 2,8 | Noble gas (full outer shell) |
| Na | 11 | 2,8,1 | |
| Mg | 12 | 2,8,2 | |
| Al | 13 | 2,8,3 | |
| Si | 14 | 2,8,4 | |
| P | 15 | 2,8,5 | |
| S | 16 | 2,8,6 | |
| Cl | 17 | 2,8,7 | |
| Ar | 18 | 2,8,8 | Noble gas (full outer shell) |
| K | 19 | 2,8,8,1 | |
| Ca | 20 | 2,8,8,2 | |

### Electronic Configuration and the Periodic Table
- Number of **shells** = period number (row)
- Number of **outer shell electrons** = group number
- Elements in the same **group** have the same number of outer electrons → similar chemical properties
- **Noble gases** (Group 0/18) have full outer shells → very unreactive

### Ion Formation from Electronic Configuration

| Atom | Configuration | Ion | Configuration |
|------|--------------|-----|--------------|
| Na (2,8,1) | Loses 1e⁻ | Na⁺ | 2,8 |
| Mg (2,8,2) | Loses 2e⁻ | Mg²⁺ | 2,8 |
| Cl (2,8,7) | Gains 1e⁻ | Cl⁻ | 2,8,8 |
| O (2,6) | Gains 2e⁻ | O²⁻ | 2,8 |

> 💡 Ions achieve the **noble gas electronic configuration** (full outer shell = stable)

## Key Definitions

| Term | Definition |
|------|-----------|
| Atomic number | Number of protons in an atom |
| Mass number | Total number of protons + neutrons |
| Isotopes | Atoms of same element with same atomic number but different mass numbers |
| Relative atomic mass | Weighted average mass of naturally occurring isotopes relative to ¹²C |
| Electronic configuration | Arrangement of electrons in shells around the nucleus |
