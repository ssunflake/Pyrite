## 📋 SMILES Syntax Cheat Sheet

| Symbol | Meaning | Example |
|--------|---------|---------|
| `C` | Carbon atom | `C` = methane (CH₄) |
| `N` | Nitrogen | `N` = ammonia (NH₃) |
| `O` | Oxygen | `O` = water (H₂O) |
| `S` | Sulfur | `S` = hydrogen sulfide |
| `=` | Double bond | `C=C` = ethene |
| `#` | Triple bond | `C#C` = ethyne |
| `()` | Branch | `CC(C)C` = methylpropane |
| `[atom]` | Formal charge/special | `[NH4+]` = ammonium |
| `.` | Disconnected ions | `[Na+].[Cl-]` = NaCl |
| `c` (lowercase) | Aromatic carbon | `c1ccccc1` = benzene |
| `@@` / `@` | Stereochemistry (chirality) | Glucose |
