## 22.1 Tests to Identify Common Cations

### Flame Tests (for metal ions)

| Metal ion | Flame colour |
|-----------|-------------|
| Lithium (Li⁺) | Crimson red |
| Sodium (Na⁺) | Persistent yellow |
| Potassium (K⁺) | Lilac / pale purple |
| Calcium (Ca²⁺) | Brick red / orange-red |
| Barium (Ba²⁺) | Pale/apple green |
| Copper (Cu²⁺) | Blue-green / verdigris |

**Method:** Clean nichrome wire with conc. HCl; dip in sample; hold in Bunsen flame; observe colour.

---

### Testing Cations with Sodium Hydroxide Solution (NaOH)

Adding NaOH(aq) to a solution of the unknown cation — look for precipitate colour and whether it dissolves in excess NaOH.

| Ion | Precipitate colour | Dissolves in excess NaOH? |
|-----|-------------------|--------------------------|
| Al³⁺ | White | Yes (forms colourless aluminate solution) |
| Ca²⁺ | White | No |
| Mg²⁺ | White | No |
| Zn²⁺ | White | Yes (forms colourless zincate) |
| Fe²⁺ | Green | No |
| Fe³⁺ | Red-brown | No |
| Cu²⁺ | Blue | No |
| Pb²⁺ | White | Yes |
| Mn²⁺ | Off-white/buff | No |
| NH₄⁺ | No precipitate; pungent gas (NH₃) produced on warming | N/A |

**Ionic equations for hydroxide precipitates:**
$$\text{Cu}^{2+}(aq) + 2\text{OH}^-(aq) \rightarrow \text{Cu(OH)}_2(s) \text{ (blue ppt)}$$
$$\text{Fe}^{3+}(aq) + 3\text{OH}^-(aq) \rightarrow \text{Fe(OH)}_3(s) \text{ (red-brown ppt)}$$
$$\text{Fe}^{2+}(aq) + 2\text{OH}^-(aq) \rightarrow \text{Fe(OH)}_2(s) \text{ (green ppt)}$$
$$\text{Al}^{3+}(aq) + 4\text{OH}^-(aq) \rightarrow [\text{Al(OH)}_4]^-(aq) \text{ (excess NaOH)}$$

**Testing for NH₄⁺:**
- Add NaOH(aq), warm gently
- NH₄⁺ + OH⁻ → NH₃(g) + H₂O
- Pungent smell; turns damp red litmus paper blue

## 22.2 Tests to Identify Common Anions

### Carbonate (CO₃²⁻)
**Test:** Add dilute HCl (or any dilute acid)
**Observation:** Effervescence; colourless gas produced
**Confirm:** Bubble gas through limewater → turns milky/white (CO₂)
$$\text{CO}_3^{2-}(aq) + 2\text{H}^+(aq) \rightarrow \text{H}_2\text{O}(l) + \text{CO}_2(g)$$

### Sulfate (SO₄²⁻)
**Test:** Acidify with dilute HCl, then add BaCl₂(aq) (or Ba(NO₃)₂)
**Observation:** White precipitate (BaSO₄) forms immediately; insoluble in excess acid
$$\text{Ba}^{2+}(aq) + \text{SO}_4^{2-}(aq) \rightarrow \text{BaSO}_4(s) \downarrow \text{ (white)}$$

> ⚠️ Must acidify first (with HCl, not H₂SO₄!) to dissolve any carbonate (which would also give a white precipitate with Ba²⁺).

### Sulfite (SO₃²⁻)
**Test:** Add dilute HCl → gas produced; test with acidified K₂Cr₂O₇ solution
**Observation:** Orange dichromate turns green (SO₂ reduces Cr⁶⁺ → Cr³⁺)

### Halides (Cl⁻, Br⁻, I⁻)
**Test:** Acidify with dilute HNO₃, then add AgNO₃(aq)

| Ion | Precipitate colour | Solubility in dilute NH₃ | Solubility in concentrated NH₃ |
|-----|---------------------|--------------------------|-------------------------------|
| Cl⁻ | White (AgCl) | Dissolves | Dissolves |
| Br⁻ | Pale yellow (AgBr) | Insoluble | Dissolves |
| I⁻ | Yellow (AgI) | Insoluble | Insoluble |

> ⚠️ Must acidify with dilute HNO₃ (not HCl — would interfere!) to remove carbonates etc. before adding AgNO₃.

### Nitrate (NO₃⁻)
**Test (brown ring):** Add iron(II) sulfate solution, then carefully add concentrated H₂SO₄
**Observation:** Brown ring at junction of layers
(Or: add NaOH + Al foil, warm → NH₃ produced; test with damp red litmus)

### Phosphate (PO₄³⁻)
**Test:** Add ammonium molybdate solution and acidify with HNO₃, warm
**Observation:** Yellow precipitate

## 22.3 Tests to Identify Common Gases

| Gas | Test | Positive result |
|-----|------|----------------|
| **Hydrogen (H₂)** | Burning splint held to mouth of tube | Burns with a 'squeaky pop' |
| **Oxygen (O₂)** | Glowing splint held at mouth of tube | Splint relights |
| **Carbon dioxide (CO₂)** | Bubble through limewater (Ca(OH)₂ solution) | Turns milky/white (CaCO₃ precipitate) |
| **Ammonia (NH₃)** | Damp red litmus paper at mouth of tube | Turns blue (alkaline) |
| | Hold near HCl(aq) | White fumes (NH₄Cl) |
| **Chlorine (Cl₂)** | Damp blue litmus paper | Turns red then bleaches white |
| | Bleaches litmus/dye | Colour removed |
| **Hydrogen chloride (HCl)** | Damp blue litmus | Turns red (acidic) |
| | Hold near NH₃ | White fumes (NH₄Cl) |
| **Sulfur dioxide (SO₂)** | Acidified K₂Cr₂O₇ paper | Orange → green |
| | Damp blue litmus | Turns red |
| **Bromine (Br₂)** | Damp litmus | Bleaches |
| | Starch-iodide paper | Doesn't turn blue (distinguish from Cl₂) |
| **Water vapour (H₂O)** | Anhydrous CuSO₄ (white powder) | Turns blue (hydrated CuSO₄) |
| | Anhydrous cobalt chloride paper | Blue → pink |

## 22.4 Quantitative Analysis: Acid–Base Titrations

> **Titration** is a technique used to find the exact volume of one solution that reacts completely with a known volume of another solution.

### Equipment
- **Burette** (contains solution of known concentration — usually acid)
- **Pipette** (delivers fixed accurate volume — usually alkali)
- **Conical flask** (contains solution + indicator)
- **White tile** (background for indicator colour change)
- Indicator: phenolphthalein (colourless → pink) or methyl orange (yellow → orange/red)

### Procedure

1. Rinse burette with solution to be used (remove distilled water traces)
2. Fill burette; record initial volume
3. Rinse pipette, transfer **exactly** 25.0 cm³ of alkali to conical flask
4. Add 2–3 drops of indicator
5. Add acid from burette dropwise near endpoint; swirl
6. Stop when indicator **permanently** changes colour (endpoint)
7. Record final burette volume → titre = final − initial
8. Repeat to get concordant results (within 0.1 cm³ of each other)
9. Average concordant titres

### Calculations

**Example:** 25.0 cm³ of NaOH is titrated with 0.100 mol/dm³ HCl. Titre = 20.0 cm³.

$$\text{NaOH}(aq) + \text{HCl}(aq) \rightarrow \text{NaCl}(aq) + \text{H}_2\text{O}(l)$$

1. n(HCl) = c × V = 0.100 × (20.0/1000) = **0.00200 mol**
2. Mole ratio HCl:NaOH = 1:1 → n(NaOH) = **0.00200 mol**
3. c(NaOH) = n/V = 0.00200 / (25.0/1000) = **0.0800 mol/dm³**

### Indicators and Their End Points

| Indicator | Acidic colour | Neutral | Alkaline colour | Best for |
|-----------|--------------|---------|-----------------|----------|
| Methyl orange | Red | Orange | Yellow | Strong acid/strong base; strong acid/weak base |
| Phenolphthalein | Colourless | Colourless | Pink | Strong acid/strong base; weak acid/strong base |
| Litmus | Red | Purple | Blue | Not precise enough for titrations |
| Universal indicator | Red | Green | Violet | Not for titrations (gradual change) |

### Back Titration
Used when: direct titration is difficult (e.g. substance is insoluble; no suitable indicator).

**Method:**
1. Add **excess** known reagent to sample
2. Titrate **remaining excess** reagent with standard solution
3. Amount that reacted = original excess − excess remaining

## Key Definitions

| Term | Definition |
|------|-----------|
| Flame test | Observation of characteristic colour produced when metal compound is heated in a flame |
| Precipitate | Insoluble solid formed when two solutions are mixed |
| Titration | Technique to determine exact volume of one solution reacting with another |
| Endpoint | Point in titration where indicator changes colour permanently |
| Titre | Volume of solution delivered from burette in a titration |
| Concordant results | Titres agreeing within 0.1 cm³ |
| Limewater | Calcium hydroxide solution Ca(OH)₂; turns milky in CO₂ |
