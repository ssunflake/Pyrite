## 10.1 Combustion, Oxidation and Reduction

### Definitions — Progression of Understanding

| Level | Oxidation | Reduction |
|-------|-----------|-----------|
| Simple | Addition of oxygen OR removal of hydrogen | Removal of oxygen OR addition of hydrogen |
| Ionic | Loss of electrons | Gain of electrons |
| Oxidation state | Increase in oxidation state | Decrease in oxidation state |

> **OIL RIG** — Oxidation Is Loss (of electrons), Reduction Is Gain (of electrons)

### Oxidising and Reducing Agents

| Term | Definition | What happens to it? |
|------|-----------|-------------------|
| **Oxidising agent** | Causes oxidation in another substance | Gets **reduced** (gains electrons) |
| **Reducing agent** | Causes reduction in another substance | Gets **oxidised** (loses electrons) |

**Common oxidising agents:** O₂, Cl₂, KMnO₄, H₂O₂, concentrated HNO₃, F₂, MnO₂

**Common reducing agents:** H₂, C, CO, metals (Na, Mg, Zn, Fe), SO₂

### Combustion

$$\text{C}(s) + \text{O}_2(g) \rightarrow \text{CO}_2(g) \quad \text{(complete)}$$
$$2\text{C}(s) + \text{O}_2(g) \rightarrow 2\text{CO}(g) \quad \text{(incomplete — limited O}_2\text{)}$$

**Hydrocarbons (complete):** CₓHᵧ + O₂ → CO₂ + H₂O
**Hydrocarbons (incomplete):** Produces CO and/or soot (C)

## 10.2 Redox Reactions

### Oxidation States (Oxidation Numbers)

Rules for assigning oxidation states:

| Rule | Oxidation state |
|------|----------------|
| Elements in their standard state | 0 |
| Simple monoatomic ions | Equal to ionic charge (Na⁺ = +1, Fe²⁺ = +2) |
| Oxygen in compounds | −2 (except peroxides: −1, OF₂: +2) |
| Hydrogen in compounds | +1 (except metal hydrides: −1) |
| Fluorine always | −1 |
| Sum in neutral compound | = 0 |
| Sum in polyatomic ion | = charge of ion |

### Examples of Oxidation States

| Compound | Element | Working | Oxidation state |
|----------|---------|---------|----------------|
| H₂O | O = −2, H = +1 | H: 2(+1) + O: −2 = 0 ✓ | H = +1, O = −2 |
| H₂SO₄ | S | 2(+1) + S + 4(−2) = 0 → S = +6 | S = +6 |
| HNO₃ | N | +1 + N + 3(−2) = 0 → N = +5 | N = +5 |
| MnO₄⁻ | Mn | Mn + 4(−2) = −1 → Mn = +7 | Mn = +7 |
| Fe₂O₃ | Fe | 2Fe + 3(−2) = 0 → Fe = +3 | Fe = +3 |
| Cr₂O₇²⁻ | Cr | 2Cr + 7(−2) = −2 → Cr = +6 | Cr = +6 |

### Identifying Redox Reactions

A reaction is **redox** if oxidation states **change**. Identify which species is oxidised/reduced.

**Example:**
$$\text{Zn}(s) + \text{CuSO}_4(aq) \rightarrow \text{ZnSO}_4(aq) + \text{Cu}(s)$$

- Zn: 0 → +2 (oxidation state increases) → **Zn is oxidised** (it is the reducing agent)
- Cu: +2 → 0 (oxidation state decreases) → **Cu²⁺ is reduced** (CuSO₄ is the oxidising agent)

### Balancing Redox Equations (Half-equation method)

1. Write separate half-equations for oxidation and reduction
2. Balance atoms and charges in each (add H⁺, H₂O, e⁻ as needed)
3. Multiply to equalise electrons
4. Add half-equations together and cancel

**Example:** MnO₄⁻ + Fe²⁺ in acid

Reduction: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
Oxidation: Fe²⁺ → Fe³⁺ + e⁻

Multiply oxidation × 5:  5Fe²⁺ → 5Fe³⁺ + 5e⁻

Add: **MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺**

### Important Redox Reactions

| Reaction | Oxidised | Reduced |
|----------|---------|---------|
| Fe rusting: 4Fe + 3O₂ → 2Fe₂O₃ | Fe (0 → +3) | O₂ (0 → −2) |
| Zn + 2HCl → ZnCl₂ + H₂ | Zn (0 → +2) | H (+ 1 → 0) |
| 2Mg + O₂ → 2MgO | Mg (0 → +2) | O₂ (0 → −2) |
| CuO + H₂ → Cu + H₂O | H₂ (0 → +1) | Cu (+2 → 0) |

## Key Definitions

| Term | Definition |
|------|-----------|
| Oxidation | Loss of electrons / increase in oxidation state |
| Reduction | Gain of electrons / decrease in oxidation state |
| Oxidising agent | Accepts electrons; is itself reduced |
| Reducing agent | Donates electrons; is itself oxidised |
| Oxidation state | A number representing the charge an atom would have if bonding were fully ionic |
| Redox reaction | A reaction involving simultaneous oxidation and reduction |
