##  Key Frameworks at a Glance

| Framework | Used For | Stands For |
|-----------|---------|-----------|
| **PEE** | Comprehension + argument paragraphs | Point → Evidence → Explanation |
| **PEEL** | Extended analysis paragraphs | Point → Evidence → Explanation → Link |
| **MQEP** | Language analysis paragraphs | Method → Quotation → Effect → Purpose |
| **AFOREST** | Persuasive writing techniques | Anecdote, Facts, Opinions, Rhetorical Q, Emotive language, Statistics, Triplets |
| **FANBOYS** | Coordinating conjunctions | For, And, Nor, But, Or, Yet, So |
| **TIPTOP** | When to start a new paragraph | Time, Idea, Place, Topic, Opinion, Person |

>  **Exam tip:** Always read the full text carefully at least once before attempting any questions. Annotate as you read — underline key phrases, note the tone of each paragraph, and mark any language techniques you spot. This two-minute investment will significantly improve the accuracy and depth of every answer you write.
