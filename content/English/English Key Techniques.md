## 1. Reading Question Types and Mark Allocations

| Question Type | What It Asks | Typical Marks | Strategy |
|--------------|-------------|---------------|---------|
| Explicit comprehension | Surface-level fact retrieval | 1–2 per point | Find and paraphrase directly from text |
| Implicit / inferential | Reading between the lines | 2–3 per point | Infer + support with textual evidence |
| Language analysis | How / why the writer uses language | 3–5+ | Technique → Quote → Effect → Purpose |
| Summary | Identify and condense key points | 10 total | Select, rephrase in own words, link |
| Evaluation | Judge how effectively the writer achieves their purpose | 5–8 | PEE + personal judgement + justification |

---

## 2. PEE / PEEL Paragraph Structure

```
POINT      →  Make your main idea or argument clearly
EVIDENCE   →  Quote or reference the specific part of the text
EXPLAIN    →  Explain how the evidence supports your point
LINK       →  Link back to the question / develop further (PEEL only)
```

**Worked Example:**
> **P** — The writer creates a sense of danger in the opening paragraph.
> **E** — He describes the darkness as something that "swallowed the path whole."
> **E** — The metaphor of darkness "swallowing" the path is predatory and consuming, suggesting the character is at risk and has lost all sense of direction.
> **L** — This immediately establishes a threatening atmosphere and compels the reader to read on, anxious for the character's safety.

## 4. Summary Writing — Step-by-Step Checklist

- [ ] Read the source text fully at least once
- [ ] Re-read and annotate / underline only the key relevant points
- [ ] Check exactly what the question is asking you to summarise
- [ ] Select 6–10 distinct, non-overlapping points directly relevant to the question
- [ ] Rephrase each point in your own words (synonyms + restructured sentences)
- [ ] Write an opening sentence that addresses the question directly
- [ ] Use linking phrases to connect points smoothly into flowing prose
- [ ] Stay within the target word count (typically 100–200 words)
- [ ] Do NOT include your own opinions or add extra information
- [ ] Check: have you copied anything word-for-word? If yes, rephrase it

## 5. Directed Writing Forms — Format Conventions

| Form | Key Layout Features | Opening | Closing |
|------|-------------------|---------|---------|
| Formal letter | Sender address (top right), date, recipient address (left) | Dear Sir/Madam | Yours faithfully |
| Informal letter | Less rigid layout, friendly tone, contractions fine | Dear [First Name] | Yours sincerely / Love / Best wishes |
| Speech | Greeting, rhetorical devices, direct address to audience | "Good morning, ladies and gentlemen" | "Thank you for listening" |
| Newspaper article | Headline in CAPS/bold, byline (By [Name]), 5 Ws in opening, subheadings, quotes | Bold headline | No formal sign-off |
| Magazine article | Engaging headline, subheadings, chatty or formal tone | Hook sentence | Strong closing thought |
| Report | Title, Introduction, Findings, Recommendations, formal tone, may number sections | "Introduction:" | "Recommendations:" |
| Interview | Q&A format, interviewer labelled, clear purposeful questions | "Thank you for joining us today..." | "Thank you for your time." |
| Email | Subject line, appropriate greeting | Subject: [Topic] / Dear... | Regards / Best wishes |

## 6. AFOREST — Persuasive Techniques Mnemonic

| Letter | Technique | Definition | Example |
|--------|-----------|-----------|---------|
| **A** | Anecdote | Short personal or relevant story to connect emotionally | "When I was twelve, I watched our local park disappear under concrete overnight..." |
| **F** | Facts | Verifiable true statements that build credibility | "Over 8 million tonnes of plastic enter our oceans every single year." |
| **O** | Opinions | The writer's viewpoint, confidently expressed | "It is simply unacceptable that we continue to ignore this crisis." |
| **R** | Rhetorical questions | Questions asked for effect; no genuine answer needed | "How many more lives must be lost before we decide to act?" |
| **E** | Emotive language | Words that trigger a strong emotional response | "innocent victims," "devastating neglect," "heartbreaking scenes" |
| **S** | Statistics | Numerical data to lend authority to an argument | "70% of teenagers report feeling seriously stressed about their futures." |
| **T** | Triplets (rule of three) | Three words or phrases in a row for rhythm and emphasis | "We must act quickly, decisively, and fearlessly." |

## 7. Literary Devices — Quick List (18 Devices)

| Device | One-Line Definition |
|--------|-------------------|
| Simile | Comparison using 'like' or 'as' |
| Metaphor | Direct comparison — one thing IS another |
| Extended metaphor | A metaphor sustained across a whole passage |
| Personification | Giving human qualities to a non-human thing |
| Alliteration | Repetition of the same consonant sound at the start of nearby words |
| Assonance | Repetition of vowel sounds within nearby words |
| Onomatopoeia | A word that sounds like what it describes (buzz, crash, hiss) |
| Hyperbole | Deliberate, dramatic exaggeration for effect |
| Understatement / litotes | Deliberately playing down something significant |
| Verbal irony | Saying the opposite of what you mean |
| Dramatic irony | The audience knows something the character does not |
| Sarcasm | Sharp, cutting irony used to mock or criticise |
| Pathetic fallacy | Environment / weather reflecting emotion |
| Foreshadowing | Hints or clues that suggest future events |
| Juxtaposition | Placing contrasting ideas side by side to highlight difference |
| Anaphora | Repeating words at the beginning of successive clauses |
| Oxymoron | Combining contradictory terms ("bitter sweet," "deafening silence") |
| Symbolism | Using an object or event to represent an abstract idea |
| Euphemism | A mild phrase used instead of something blunt or harsh |

## 8. Punctuation Effects Table

| Punctuation | Effect / Use in Writing |
|-------------|------------------------|
| Full stop (.) | Finality; very short sentences create abruptness or shock |
| Comma (,) | Controlled pause; separates list items or clauses; controls pace |
| Semicolon (;) | Links two related independent clauses; creates balance and sophistication |
| Colon (:) | Introduces a list, explanation, or quotation; builds anticipation |
| Dash (—) | Dramatic pause, sudden afterthought, or emphasis; can feel informal or urgent |
| Ellipsis (...) | Trailing off, hesitation, unfinished thought, suspense |
| Exclamation mark (!) | Strong emotion — shock, excitement, urgency, anger |
| Question mark (?) | Direct question; if rhetorical, challenges or engages the reader |
| Parentheses ( ) | Aside or extra information; creates intimate, conversational tone |
| Apostrophe (') | Omission (don't) or possession (the cat's); misuse is a common error |
| Quotation marks (" ") | Encloses speech or titles; can create ironic distance ("so-called") |
| Hyphen (-) | Joins compound modifiers (e.g., "well-known"; "fast-paced") |

## 10. Register — Formal vs Informal

| Feature | Formal Register | Informal Register |
|---------|----------------|------------------|
| Vocabulary | Sophisticated, precise, technical | Simple, colloquial, everyday |
| Contractions | Avoided ("do not," "cannot") | Common ("don't," "can't") |
| Sentence structure | Complex, varied, controlled | Simple, conversational, relaxed |
| Tone | Neutral, measured, respectful | Friendly, personal, warm |
| Pronouns | Often impersonal ("one," "it is argued") | Direct ("I," "you," "we") |
| Slang | Never used | May be used appropriately |
| Humour | Rare, subtle if present | Natural and frequent |
| Typical use | Reports, formal letters, academic essays | Informal letters, blogs, texts |
