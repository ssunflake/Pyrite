## DNA, Genes and Chromosomes

| Term | Definition |
|------|-----------|
| **DNA** | Deoxyribonucleic acid — double-stranded molecule carrying genetic information |
| **Gene** | A section of DNA that codes for a specific protein (sequence of bases) |
| **Chromosome** | A long molecule of DNA associated with proteins (histones); humans have 46 (23 pairs) |
| **Genome** | The complete set of genetic information in an organism |
| **Allele** | A version of a gene (different alleles of the same gene) |
| **Locus** | The specific position of a gene on a chromosome |

### Human Chromosomes
- 46 chromosomes in **23 homologous pairs**
- **22 pairs:** autosomes (same in males and females)
- **1 pair:** sex chromosomes
  - **Female: XX**
  - **Male: XY**

---

## DNA Structure (Recap from Ch 4)

- Double helix made of **nucleotides**
- Each nucleotide: deoxyribose sugar + phosphate group + nitrogenous base
- Bases: **A, T, G, C** (A pairs with T; G pairs with C)
- Held together by **hydrogen bonds** between complementary bases

---

## The Genetic Code

- A **codon** = sequence of **3 bases** on mRNA coding for one amino acid
- 4 bases × 3 positions = 64 possible codons (more than enough for 20 amino acids)
- Code is **degenerate** (multiple codons can code for same amino acid)
- Code is **universal** (same in all organisms — basis of genetic engineering)

---

## Protein Synthesis

### Step 1: Transcription (in nucleus)
1. DNA unwinds and unzips (hydrogen bonds break)
2. **mRNA** (messenger RNA) synthesised using one strand of DNA as template
3. **RNA polymerase** adds complementary RNA bases (A→U, T→A, G→C, C→G)
4. mRNA detaches and moves out of nucleus through nuclear pore

### Step 2: Translation (at ribosomes in cytoplasm)
1. mRNA attaches to ribosome
2. **tRNA** molecules carry amino acids to ribosome
3. Each tRNA has an **anticodon** that pairs with a codon on mRNA
4. Amino acids joined by **peptide bonds**
5. Polypeptide chain grows → folds into protein with specific 3D shape

## Mutations

> A **mutation** is a change in the sequence of bases in DNA.

### Types of Mutations
| Type | Description | Effect |
|------|-------------|--------|
| **Substitution** | One base replaced by another | May change one amino acid; might have no effect if degenerate code |
| **Insertion** | Extra base added | Frameshift — changes ALL subsequent codons → severe effect |
| **Deletion** | Base removed | Frameshift — changes ALL subsequent codons → severe effect |

### Causes of Mutations (Mutagens)
- **Radiation:** UV light, X-rays, gamma rays
- **Chemical mutagens:** tobacco smoke, some dyes
- **Viruses**
- Mutations occur **spontaneously** (random errors in DNA replication)

### Effects of Mutations
- Most mutations are **neutral** (degenerate code, non-coding DNA)
- Some are **harmful** → cancer, genetic disorders
- Very rarely: **beneficial** → natural selection

## Genetics and Inheritance

### Key Terms
| Term | Definition |
|------|-----------|
| **Genotype** | Genetic makeup of an organism (alleles it carries) |
| **Phenotype** | Observable characteristics (result of genotype + environment) |
| **Homozygous** | Both alleles the same (AA or aa) |
| **Heterozygous** | Two different alleles (Aa) |
| **Dominant** | Allele expressed in both homozygous dominant and heterozygous — written as capital letter (A) |
| **Recessive** | Allele only expressed when homozygous — written as lower case (a) |
| **Codominance** | Both alleles expressed equally in heterozygote |

### Monohybrid Crosses
- Use **Punnett squares** to predict offspring ratios

**Example: Tall (T) × short (t) plant:**

|  | T | t |
|--|---|---|
| **T** | TT | Tt |
| **t** | Tt | tt |

Result: 3 tall : 1 short (ratio 3:1)

### Dihybrid Cross (Two traits)
- Consider two genes independently (if on different chromosomes)
- Standard 9:3:3:1 ratio for TtRr × TtRr

## Sex Determination

- Father contributes either X or Y to offspring
- 50% chance of male or female each pregnancy

|  | X | Y |
|--|---|---|
| **X** | XX (female) | XY (male) |
| **X** | XX (female) | XY (male) |

## Sex-Linked Inheritance

- Some genes carried on **X chromosome** with no corresponding allele on Y
- Conditions: colour blindness, haemophilia

**Example — colour blindness (X-linked recessive):**
- $X^N$ = normal vision (dominant)
- $X^n$ = colour blind (recessive)
- Females need **two** copies of $X^n$ to be colour blind ($X^nX^n$)
- Males need only **one** ($X^nY$) — hence more common in males

---

## Codominance

- Neither allele is dominant over the other
- Both expressed simultaneously in heterozygote

**Example — ABO blood groups:**
- Alleles: $I^A$, $I^B$, $i$
- $I^A$ and $I^B$ are codominant; $i$ is recessive

| Genotype | Blood group |
|----------|------------|
| $I^AI^A$ or $I^Ai$ | A |
| $I^BI^B$ or $I^Bi$ | B |
| $I^AI^B$ | AB (both antigens present) |
| $ii$ | O |

## Sickle Cell Anaemia

- Autosomal recessive condition
- Mutation in haemoglobin gene → abnormal haemoglobin → RBCs become sickle-shaped
- Sickle cells: block capillaries, break down quickly → anaemia, pain, organ damage
- **Carriers** ($HbA Hbs$): mostly healthy; some protection against malaria (advantage in malaria regions)
