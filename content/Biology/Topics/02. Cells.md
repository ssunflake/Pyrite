## The Cell Theory
- All living organisms are made of **cells**
- The cell is the basic unit of life
- All cells come from pre-existing cells

---

## Animal Cell vs Plant Cell

| Feature | Animal Cell | Plant Cell |
|---------|------------|-----------|
| **Cell membrane** | ✅ | ✅ |
| **Nucleus** | ✅ | ✅ |
| **Cytoplasm** | ✅ | ✅ |
| **Mitochondria** | ✅ | ✅ |
| **Ribosomes** | ✅ | ✅ |
| **Cell wall** | ❌ | ✅ (cellulose) |
| **Large vacuole** | ❌ (small/absent) | ✅ (permanent, central) |
| **Chloroplasts** | ❌ | ✅ (in green cells) |
| **Regular shape** | ❌ (varies) | ✅ (more regular) |

## Cell Organelles — Functions

| Organelle | Function |
|-----------|---------|
| **Cell membrane** | Controls what enters and exits the cell (selectively permeable) |
| **Nucleus** | Contains DNA; controls cell activities; contains instructions for protein synthesis |
| **Cytoplasm** | Site of many chemical reactions; where organelles are suspended |
| **Mitochondria** | Site of aerobic respiration; produces ATP (energy) |
| **Ribosomes** | Site of protein synthesis |
| **Cell wall** (plant) | Provides support and shape; made of cellulose; fully permeable |
| **Vacuole** (plant) | Contains cell sap; maintains turgor pressure; storage |
| **Chloroplasts** (plant) | Site of photosynthesis; contains chlorophyll |
| **Endoplasmic reticulum** | Transport network within cell; rough ER has ribosomes |
| **Golgi apparatus** | Packages and secretes proteins |
| **Lysosomes** | Contain digestive enzymes; break down waste material |

## Prokaryotic vs Eukaryotic Cells

| Feature | Prokaryotic (e.g. bacteria) | Eukaryotic (animals, plants, fungi) |
|---------|---------------------------|-------------------------------------|
| Nucleus | ❌ (no membrane-bound nucleus; DNA free in cytoplasm) | ✅ (membrane-bound nucleus) |
| Size | Smaller (~1–10 μm) | Larger (~10–100 μm) |
| Membrane-bound organelles | ❌ | ✅ |
| Cell wall | ✅ (not cellulose) | Plants/fungi ✅; Animals ❌ |
| Ribosomes | ✅ (smaller 70S) | ✅ (larger 80S) |
| DNA | Circular, no histones | Linear chromosomes, with histones |

## Specialised Cells

### Animal Cells
| Cell | Adaptations | Function |
|------|------------|---------|
| **Red blood cell** | Biconcave disc (large SA); no nucleus (more room for Hb); haemoglobin | Carry oxygen |
| **Sperm cell** | Long tail (flagellum) for movement; many mitochondria; acrosome (enzymes to penetrate egg) | Fertilisation |
| **Nerve cell (neurone)** | Long axon; myelin sheath (insulation); many dendrites | Transmit electrical impulses |
| **Ciliated cell** | Many cilia (tiny hairs) on surface | Move mucus in airways |
| **White blood cell** | Large, irregular shape; flexible membrane | Engulf pathogens (phagocytosis) |

### Plant Cells
| Cell | Adaptations | Function |
|------|------------|---------|
| **Root hair cell** | Long thin extension; large surface area; no chloroplasts | Absorb water and minerals |
| **Palisade mesophyll** | Many chloroplasts; near leaf surface; tall shape | Maximum photosynthesis |
| **Guard cell** | Curved thick/thin walls; chloroplasts; opens/closes stomata | Controls gas exchange and water loss |
| **Xylem vessel** | Dead; hollow; lignified walls; no end walls | Transport water and minerals |
| **Phloem cell** | Living; sieve plates; companion cells | Transport sugars |

## Cell Organisation

**Cell → Tissue → Organ → Organ System → Organism**

| Level | Definition | Example |
|-------|-----------|---------|
| **Cell** | Basic structural unit | Muscle cell |
| **Tissue** | Group of similar cells with the same function | Muscle tissue |
| **Organ** | Group of different tissues working together | Heart |
| **Organ system** | Group of organs working together | Circulatory system |
| **Organism** | Complete living thing | Human |

## Cell Division

### Mitosis
- Produces **two** genetically **identical** daughter cells
- Used for: growth, repair, asexual reproduction
- Cell duplicates DNA → cell divides once
- Daughter cells have same chromosome number as parent (diploid: 2n)

### Meiosis
- Produces **four** genetically **different** daughter cells
- Used for: production of gametes (sex cells)
- Cell divides **twice**
- Daughter cells have **half** the chromosome number (haploid: n)
- Produces genetic variation through crossing over and independent assortment

| Feature | Mitosis | Meiosis |
|---------|---------|---------|
| Number of divisions | 1 | 2 |
| Daughter cells | 2 | 4 |
| Genetic variation | No (identical) | Yes |
| Chromosome number | Maintained (2n→2n) | Halved (2n→n) |
| Purpose | Growth, repair | Gamete production |

## Key Definitions

| Term | Definition |
|------|-----------|
| **Organelle** | Specialised structure within a cell that carries out a specific function |
| **Tissue** | Group of similar cells performing the same function |
| **Differentiation** | Process by which cells become specialised for a particular function |
| **Stem cell** | Undifferentiated cell capable of dividing and differentiating into specialised cells |
