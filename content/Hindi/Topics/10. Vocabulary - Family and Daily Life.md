## 10.1 Family (परिवार — Parivār)

Hindi has an exceptionally rich family vocabulary — every relationship has its own specific term. Many of these are unique to South Asian languages.

| Hindi | Transliteration | English | Notes |
|-------|-----------------|---------|-------|
| पिता / बाबा / पापा | pitā / bābā / pāpā | father | पिता is formal; पापा is colloquial |
| माता / माँ / अम्मा | mātā / māṃ / ammā | mother | माता is formal; माँ is everyday |
| भाई | bhāī | brother | |
| बहन | bahan | sister | |
| दादा | dādā | paternal grandfather (father's father) | |
| दादी | dādī | paternal grandmother (father's mother) | |
| नाना | nānā | maternal grandfather (mother's father) | |
| नानी | nānī | maternal grandmother (mother's mother) | |
| चाचा | cācā | father's younger brother | |
| चाची | cācī | father's younger brother's wife | |
| ताया / ताऊ | tāyā / tāū | father's older brother | |
| ताई | tāī | father's older brother's wife | |
| मामा | māmā | mother's brother | |
| मामी | māmī | mother's brother's wife | |
| मौसा | mausā | mother's sister's husband | |

> 💡 **Tip:** Hindi distinguishes between paternal and maternal relatives (दादा/नाना, दादी/नानी) and between older/younger paternal uncles (ताया/चाचा). These terms show how important family structure is in Indian culture.

## 10.6 Sample Sentences — Daily Life

| Hindi | Transliteration | English | Tense |
|-------|-----------------|---------|-------|
| मैं हर सुबह सात बजे उठता हूँ। | maiṃ har subah sāt baje uṭhtā hūṃ. | I get up every morning at seven o'clock. | Present habitual |
| माँ रसोई में खाना बना रही हैं। | māṃ rasoī meṃ khānā banā rahī haiṃ. | Mother is cooking food in the kitchen. | Present continuous |
| हमने कल रात बिरयानी खाई। | hamne kal rāt biryānī khāī. | Last night we ate biryani. | Simple past (transitive) |
| मेरे दादाजी बगीचे में बैठते हैं। | mere dādājī bagīce meṃ baiṭhte haiṃ. | My grandfather sits in the garden. | Present habitual |
| कल हम सब मिलकर खाना खाएँगे। | kal ham sab milkar khānā khāeṃge. | Tomorrow we will all eat together. | Future tense |
| मेरी बहन को सेब बहुत पसंद है। | merī bahan ko seb bahut pasaṃd hai. | My sister really likes apples. | को-construction |
| मुझे सिर में दर्द है। | mujhe sir meṃ dard hai. | I have a headache. | को-sensation |
