## 6.1 When to Use L'Imparfait

The imparfait is used in the past for:

| Use | Description | Example |
|-----|-------------|---------|
| **Descriptions** | Weather, appearance, state of mind, background | *Il faisait beau.* (It was nice weather.) |
| **Habits / repetition** | Used to / would do regularly | *Je jouais au foot chaque semaine.* |
| **Ongoing action** | Action in progress at a specific past moment | *Je dormais quand...* |
| **States and conditions** | Existing over a period (not a specific moment) | *Elle était triste.* |
| **Polite requests** | Softens tone | *Je voulais vous demander quelque chose.* |
| **Si + imparfait** | Hypothetical conditions | *Si j'avais de l'argent...* |

---

## 6.2 Formation of L'Imparfait

**Method:** Take the **nous** form of the present tense → remove **-ons** → add imparfait endings.

### Imparfait Endings

| Pronoun | Ending |
|---------|--------|
| je | **-ais** |
| tu | **-ais** |
| il / elle / on | **-ait** |
| nous | **-ions** |
| vous | **-iez** |
| ils / elles | **-aient** |

### The Nous-Stem Method

| Verb | Nous-form (present) | Stem | Imparfait (je) |
|------|--------------------|----- |---------------|
| parler | nous parlons | parl- | je parlais |
| finir | nous finissons | finiss- | je finissais |
| vendre | nous vendons | vend- | je vendais |
| prendre | nous prenons | pren- | je prenais |
| avoir | nous avons | av- | j'avais |
| faire | nous faisons | fais- | je faisais |
| aller | nous allons | all- | j'allais |
| voir | nous voyons | voy- | je voyais |
| savoir | nous savons | sav- | je savais |
| vouloir | nous voulons | voul- | je voulais |

### parler — Full Imparfait Conjugation

| Pronoun | Form |
|---------|------|
| je | parl**ais** |
| tu | parl**ais** |
| il / elle / on | parl**ait** |
| nous | parl**ions** |
| vous | parl**iez** |
| ils / elles | parl**aient** |

### être — The Only Truly Irregular Imparfait

*Être* does NOT follow the nous-stem method. Its stem is **ét-**:

| Pronoun | Form |
|---------|------|
| je | **étais** |
| tu | **étais** |
| il / elle / on | **était** |
| nous | **étions** |
| vous | **étiez** |
| ils / elles | **étaient** |

> [!note] *Être* is the ONLY verb with an irregular imparfait stem. All other verbs — even highly irregular ones — use the *nous* present tense stem.

### Additional Conjugation Examples

**finir (imparfait):**

| je finissais | nous finissions |
|-------------|----------------|
| tu finissais | vous finissiez |
| il finissait | ils finissaient |

**faire (imparfait):**

| je faisais | nous faisions |
|-----------|--------------|
| tu faisais | vous faisiez |
| il faisait | ils faisaient |

**Note on -ger verbs:** The soft *g* spelling is kept throughout the imparfait (no special change needed since you never add just -ons after the stem):
- *manger* → je mangeais, tu mangeais, etc. (the *e* is already in the stem)

## 6.3 Uses of L'Imparfait — Detailed

### 1. Descriptions in the Past

Setting the scene — describing how things **looked, felt, or were**:
- *Il faisait beau et le soleil brillait.* — The weather was lovely and the sun was shining.
- *Elle portait une robe rouge et semblait fatiguée.* — She was wearing a red dress and seemed tired.
- *La maison **était** grande et **entourée** de jardins.* — The house was big and surrounded by gardens.
- *Je me sentais anxieux avant l'examen.* — I was feeling anxious before the exam.

### 2. Habitual / Repeated Actions (Used to / Would)

Actions that happened **regularly** or **repeatedly** in the past:
- *Quand j'**étais** petit(e), je **jouais** au foot tous les weekends.* — When I was little, I used to play football every weekend.
- *Chaque été, nous **allions** à la mer.* — Every summer, we would go to the seaside.
- *Elle **lisait** tous les soirs avant de se coucher.* — She would read every evening before going to bed.
- *D'habitude, il **prenait** le bus pour aller au collège.* — Usually, he used to take the bus to school.

### 3. Ongoing Action Interrupted by a Completed Action

The imparfait sets the **background / scene**; the passé composé describes the **interrupting event**:
- *Je **dormais** quand le téléphone **a sonné**.* — I was sleeping when the phone rang.
- *Il **regardait** la télé lorsqu'elle **est arrivée**.* — He was watching TV when she arrived.
- *Nous **marchions** dans la rue quand nous **avons vu** l'accident.* — We were walking in the street when we saw the accident.

### 4. Polite Requests (Imparfait de Politesse)

Using the imparfait softens a request, making it more polite:
- *Je **voulais** vous demander quelque chose.* — I wanted to ask you something. (= I'd like to ask you something)
- *Je **voulais** un café, s'il vous plaît.* — I would like a coffee, please.
- *Je **venais** vous informer de...* — I was coming to inform you of...

### 5. Si + Imparfait (Hypothetical — see Chapter 7)

- *Si j'**avais** le temps, j'**apprendrais** le piano.* — If I had the time, I would learn the piano.
- *Si tu **venais**, ce **serait** formidable !* — If you came, it would be wonderful!

## 6.4 Passé Composé vs Imparfait — Contrast

This is one of the most tested grammar points at IGCSE.

| Passé Composé | Imparfait |
|--------------|-----------|
| Completed action | Ongoing / background action |
| Single, specific event | Repeated / habitual action |
| Sequence of completed events | Simultaneous actions |
| Sudden change of state | Continuous state |
| Short, specific duration | Long duration / no defined end |
| Interrupts the imparfait | Sets the scene for an event |

### Time Expression Clues

| Signals Passé Composé | Signals Imparfait |
|----------------------|------------------|
| soudain / tout à coup | souvent / d'habitude |
| un jour / un soir | tous les jours / chaque jour |
| hier / la semaine dernière | quand j'étais jeune / autrefois |
| enfin / finalement | normalement / généralement |
| d'abord ... ensuite ... puis | toujours / jamais |
| une fois / deux fois | le week-end / chaque été |

### Contrast Examples (Side by Side)

| Passé Composé (event) | Imparfait (background) |
|----------------------|----------------------|
| *Il a pris son parapluie.* (He grabbed his umbrella.) | *Il pleuvait.* (It was raining.) |
| *Elle est entrée dans la pièce.* (She entered the room.) | *Tout le monde dormait.* (Everyone was sleeping.) |
| *J'ai vu un film hier.* (I watched a film yesterday.) | *Je regardais souvent des films.* (I often watched films.) |
| *Soudain, il a crié.* (Suddenly, he shouted.) | *La soirée se passait bien.* (The evening was going well.) |

> 💡 **Exam tip:** A useful test — if you can replace the French past tense with "used to" in English, it is the imparfait. If "used to" doesn't work and only a simple past ("did") fits, it is the passé composé.

## Key Definitions

| Term | Definition |
|------|-----------|
| Imparfait | Imperfect tense — used for background, descriptions, habits, and ongoing past actions |
| Radical de l'imparfait | Stem for imparfait = nous-form of present minus -ons (except être → ét-) |
| Action de fond | Background action — described with imparfait; sets the scene |
| Action principale | Main / foreground action — described with passé composé |
| Habitude | Habit / repeated action — key use of the imparfait |
| État | State / condition — key use of the imparfait (feelings, beliefs, appearances) |
| Imparfait de politesse | Use of imparfait to soften a request politely |
