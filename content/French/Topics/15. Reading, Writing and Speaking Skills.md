# PART A: Reading Skills (Paper 2)

## 15.1 Question Types

| Question type | Description | Tips |
|--------------|-------------|------|
| **True / False / Not mentioned** | Is the statement T/F/NM? | "Not mentioned" = info is simply absent; don't infer |
| **Multiple choice** | Choose A, B, C or D | Eliminate wrong answers; beware of distractors |
| **Gap-fill** | Insert a word or phrase | Skim for context clues; check grammar of the gap |
| **Short answer (in French / English)** | Write your own answer | Quote from the text carefully; watch for negatives |
| **Matching** | Match headings to paragraphs | Read all options first; work by elimination |
| **Name / Identify** | Who does what? | Scan carefully; names and pronouns are key |

---

## 15.2 Reading Strategy

Follow this order for every reading task:

1. **Read the questions first** — they tell you what to look for.
2. **Scan for key words** from the question in the text (and their synonyms).
3. **Read the relevant section carefully** — do not translate word-for-word in your head.
4. **Don't panic over unknown words** — use context, cognates, and word families.
5. **Check your answer** — does it answer the question exactly? Is it logical?
6. **Never leave a blank** — an educated guess is always better than nothing.

> 💡 **Exam tip:** For "True / False / Not Mentioned" tasks, the **"Not Mentioned"** option is the trickiest. If you can find any information in the text that relates to the statement, even vaguely, it is NOT "not mentioned". Only choose "NM" when the topic is simply never raised.

---

## 15.3 Inferring Meaning from Context

When you don't know a word:
- Look at the **sentence structure** — is it a noun, verb, or adjective?
- Look for **cognates** (words similar to English/another language): *l'architecture, excellent, probable*
- Use **prefix and suffix knowledge**: *dé-* (un-/de-), *re-* (re-), *-ment* (adverb), *-tion* (noun)
- Consider what makes **logical sense** in the context
- Look for **antonyms or synonyms** nearby in the text

## 15.4 False Friends (Faux Amis)

False friends are French words that look like English words but have different meanings. They are a common trap in reading comprehension.

| French word | Looks like | Actual French meaning | Real French for English word |
|------------|-----------|----------------------|------------------------------|
| **actuellement** | actually | currently / at the moment | *en fait / à vrai dire* |
| **sensible** | sensible | sensitive | *raisonnable / sensé(e)* |
| **formidable** | formidable | wonderful / great | *redoutable / terrifiant* |
| **car** | car | because / for (conjunction) | *la voiture* |
| **passer un examen** | to pass an exam | to sit / take an exam | *réussir un examen* |
| **chance** | chance | luck / good fortune | *occasion / possibilité* |
| **journée** | journey | day (duration) | *le voyage / le trajet* |
| **librairie** | library | bookshop | *la bibliothèque* |
| **rester** | to rest | to stay / remain | *se reposer* |
| **large** | large | wide / broad | *grand(e)* |
| **blesser** | to bless | to injure / wound | *bénir* |
| **assister (à)** | to assist | to attend (an event) | *aider / assister quelqu'un* |
| **attendre** | to attend | to wait (for) | *assister à* |
| **déception** | deception | disappointment | *la tromperie / la duperie* |
| **éventuellement** | eventually | possibly / perhaps | *finalement / à la fin* |
| **ignorer** | to ignore (deliberately) | to not know / be unaware of | *ne pas faire attention à* |
| **les informations** | information (mass noun) | the news (TV/radio) | *les renseignements* (= info) |
| **lecture** | lecture (at university) | reading | *un cours magistral / une conférence* |
| **luxurieux** | luxurious | lustful / lecherous | *luxueux/euse* |
| **sympathique** | sympathetic | nice / friendly | *compatissant(e)* |
| **prétendre** | to pretend | to claim / assert | *faire semblant de* |
| **inconvénient** | inconvenient | disadvantage / drawback | *gênant(e) / peu pratique* |
| **conférence** | conference | lecture / talk (can = conference) | *congrès / sommet* |
| **agenda** | agenda (meeting) | diary / planner | *l'ordre du jour* |
| **roman** | Roman (adj) | novel (book) | *romain(e)* |
| **stage** | stage (theatre) | work placement / training course | *la scène / l'étape* |
| **comment** | comment (remark) | how | *un commentaire / une remarque* |
| **grave** | grave (burial) | serious / severe | *la tombe / le tombeau* |
| **caution** | caution | deposit / guarantee | *la prudence / la précaution* |
| **trouble** | trouble | disorder / upset | *des problèmes / des ennuis* |

> [!note] *Stage* is particularly dangerous — *faire un stage* means "to do a work placement", NOT "to be on stage". Similarly, *la librairie* is a bookshop; *la bibliothèque* is a library.

# PART B: Writing Skills (Paper 4)

## 15.5 Types of Writing Tasks

| Task type | Register | Key features |
|-----------|---------|--------------|
| **Message (informal)** | informal / *tu* | Short, conversational, simple structures |
| **Email (informal)** | informal / *tu* | Greeting, body, sign-off; personal and direct |
| **Letter (formal)** | formal / *vous* | Standard layout, polite formulae |
| **Article / blog post** | semi-formal | Title, introduction, paragraphs, conclusion |
| **Essay (discursive)** | formal | For/against structure, opinion, justified arguments |

---

## 15.6 Formal Letter Conventions in French

**Layout:**

```
[Your name / address]                    [Date and place]
                                          Paris, le 15 octobre 2026

[Recipient's name / address]

Objet : [Subject line]

Madame, Monsieur, / Monsieur, / Madame,

[Body of letter...]

Je vous prie d'agréer, Madame/Monsieur, l'expression de mes sentiments distingués.

[Signature]
```

| French phrase | English equivalent / use |
|--------------|-------------------------|
| *Madame, Monsieur,* | Dear Sir / Madam (when unsure) |
| *À qui de droit,* | To whom it may concern |
| *Je vous écris au sujet de...* | I am writing regarding... |
| *Suite à votre annonce,...* | Further to your advertisement,... |
| *Je me permets de vous contacter...* | I am taking the liberty of contacting you... |
| *Dans l'attente de votre réponse,...* | Awaiting your reply,... |
| *Je vous prie d'agréer... mes salutations distinguées.* | Yours faithfully / sincerely |
| *Veuillez agréer...* | Please accept... (formal closing) |

## 15.8 Structuring a French Essay

A well-structured IGCSE essay / discursive piece should have:

### Introduction
- Introduce the topic
- State what you will discuss
- *De nos jours, ... est un sujet très important.* / *La question de ... fait beaucoup débat.*

### Développement (Body — 2 to 3 paragraphs)
- **Paragraph 1:** First argument / perspective + evidence / example
- **Paragraph 2:** Counter-argument / different perspective
- **Paragraph 3 (optional):** Your personal opinion, developed with reasons

### Conclusion
- Summarise key points
- State your overall opinion
- *En conclusion, ...* / *Pour conclure, ...* / *En définitive, ...*

## 15.9 Connectives for Essays

| Function | French connectives |
|---------|-------------------|
| **Addition** | de plus, en outre, également, d'ailleurs, non seulement... mais aussi |
| **Contrast** | cependant, en revanche, par contre, néanmoins, pourtant, mais, or |
| **Consequence** | donc, ainsi, par conséquent, c'est pourquoi, de ce fait |
| **Cause** | parce que, car, puisque, étant donné que, à cause de, grâce à |
| **Example** | par exemple, notamment, comme, tel que, c'est le cas de |
| **Concession** | bien que + subj, même si, malgré, quoique + subj |
| **Sequence** | d'abord, ensuite, puis, enfin, finalement, premièrement, deuxièmement |
| **Clarification** | c'est-à-dire, autrement dit, en d'autres termes |
| **Conclusion** | en conclusion, pour conclure, en définitive, en résumé, bref |

## 15.10 The ACGE Writing Check

Before submitting any written work, run through the **ACGE check**:

| Letter | French | Check |
|--------|--------|-------|
| **A** | **Accord** | Do adjectives agree with the nouns they describe? Does the past participle agree where needed? |
| **C** | **Conjugaison** | Are all verbs correctly conjugated? Is the right tense used? Do subjects and verbs agree? |
| **G** | **Genre** | Is the correct gender being used for all nouns and pronouns? |
| **É** | **Écriture** | Are accents correct? Are spellings accurate? Are capital letters used correctly (NOT for days/months)? |

**Common errors to avoid:**
- ✗ *Je suis allé au cinéma avec mes amis et on a beaucoup amusé.* → ✓ *on s'est beaucoup amusé* (reflexive verb omitted)
- ✗ *Elle a été fatiguée.* (instead of *Elle était fatiguée*) — use imparfait for states
- ✗ *J'ai besoin des informations.* → ✓ *J'ai besoin d'informations* (de, not des, after besoin de)
- ✗ *C'est plus meilleur.* → ✓ *C'est meilleur.* (never *plus* with *meilleur*)
- ✗ *Je suis né en 1 Janvier.* → ✓ *le 1er janvier* (ordinal for dates; no capital)

# PART C: Speaking Skills (Paper 3)

## 15.11 Paper 3 Format

The speaking exam lasts approximately **12 minutes** and includes:
- A **conversation** based on a card/stimulus you have prepared in advance
- A **general conversation** on topics from the syllabus

---

## 15.12 How to Extend Answers

Aim for the **OPEC structure** in every answer:

| Letter | What | Example |
|--------|------|---------|
| **O** | **Opinion** | *À mon avis, la technologie est très utile.* |
| **P** | **Point / Reason** | *...parce qu'elle nous permet de communiquer instantanément.* |
| **E** | **Example** | *Par exemple, je peux contacter mes amis partout dans le monde.* |
| **C** | **Contrast** | *Cependant, les réseaux sociaux peuvent aussi être dangereux.* |

## 15.14 Model Answer Structures for Common Questions

### "Décris ta famille." (Describe your family.)

*Dans ma famille, nous sommes [nombre] personnes. J'ai [relation + name]. Mon/ma [relation] est [adjective] et [adjective]. Ce que j'aime chez lui/elle, c'est qu'il/elle est toujours là pour moi. On s'entend très bien, même si parfois on se dispute. Dans l'ensemble, j'ai de la chance d'avoir une famille aussi [adjective].*

### "Parle-moi de ton école." (Tell me about your school.)

*Je vais au/à la [type of school] qui s'appelle [...]. Il y a environ [...] élèves. Ce que j'aime dans mon école, c'est [reason]. Ma matière préférée est [...] parce que [...]. Par contre, je trouve [...] difficile. Les professeurs sont en général [adjective]. Je pense que mon école est [adjective], bien que parfois les règles soient un peu strictes.*

### "Que fais-tu pendant ton temps libre ?" (What do you do in your free time?)

*Pendant mon temps libre, j'aime surtout [...]. Je le fais [frequency] parce que c'est [adjective]. Je fais aussi [...] avec [qui]. À l'avenir, j'aimerais essayer [...] parce que ça m'a l'air passionnant. En revanche, je n'aime pas du tout [...] car c'est [adjective].*

### "Qu'est-ce que tu veux faire après les examens ?" (What do you want to do after your exams?)

*Après les examens, j'espère [aller à l'université / faire un apprentissage / travailler / voyager]. Je voudrais étudier [...] parce que [...]. À l'avenir, mon rêve serait de devenir [...]. C'est un métier que je trouve [adjective] parce que [...]. Je sais que ce sera difficile, mais je suis déterminé(e) à réussir.*

## Key Definitions

| Term | Definition |
|------|-----------|
| Faux ami | False friend — a French word that resembles an English word but has a different meaning |
| Registre | Register — the level of formality in language (formal / informal) |
| Tutoyer | To address someone using *tu* (informal) |
| Vouvoyer | To address someone using *vous* (formal) |
| La lettre formelle | Formal letter — with specific layout and polite formulae |
| Le courriel / l'email | Email |
| La dissertation | Essay / discursive piece |
| ACGE | Writing check: Accord, Conjugaison, Genre, Écriture |
| OPEC | Speaking structure: Opinion, Point/Reason, Example, Contrast |
| Les connecteurs | Connectives — linking words and phrases used to structure arguments |
| L'introduction | Introduction — opening paragraph of an essay |
| La conclusion | Conclusion — closing paragraph of an essay |
| Un faux départ | False start — beginning a sentence, stopping, and starting again (natural in speaking) |
