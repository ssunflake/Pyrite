# PART A: Le Futur Simple

## 7.1 When to Use Le Futur Simple

| Use | Example | English |
|-----|---------|---------|
| Definite future plans | *Demain, je travaillerai.* | Tomorrow, I will work. |
| Predictions | *Il fera beau ce weekend.* | The weather will be good this weekend. |
| After *quand / lorsque / dès que / aussitôt que / tant que* | *Quand j'aurai 18 ans, je pourrai voter.* | When I am 18, I will be able to vote. |
| Formal/written context | — | More certain / formal than futur proche |

> ⚠️ **Note — Critical rule:** After time conjunctions (*quand, lorsque, dès que, aussitôt que, tant que*), French uses the **future tense** where English uses the present. This is a very common IGCSE error:
> - English: "When I **arrive**..." → French: "Quand j'**arriverai**..."
> - English: "As soon as she **finishes**..." → French: "Dès qu'elle **finira**..."

---

## 7.2 Formation of Le Futur Simple

**Formula:** Infinitive (for -RE verbs: drop the final -e) + future endings

### Future Endings

| Pronoun | Ending |
|---------|--------|
| je | **-ai** |
| tu | **-as** |
| il / elle / on | **-a** |
| nous | **-ons** |
| vous | **-ez** |
| ils / elles | **-ont** |

### parler — futur simple

| je parlerai | nous parlerons |
|------------|---------------|
| tu parleras | vous parlerez |
| il / elle / on parlera | ils / elles parleront |

### finir — futur simple

| je finirai | nous finirons |
|-----------|--------------|
| tu finiras | vous finirez |
| il finira | ils finiront |

### vendre — futur simple (drop the final -e of -re verbs)

| je vendrai | nous vendrons |
|-----------|--------------|
| tu vendras | vous vendrez |
| il vendra | ils vendront |

## 7.3 Irregular Future Stems

These stems must be memorised — but they are the **same stems used for the conditional**:

| Verb | Irregular Stem | Je form | English |
|------|---------------|---------|---------|
| être | **ser-** | je serai | I will be |
| avoir | **aur-** | j'aurai | I will have |
| aller | **ir-** | j'irai | I will go |
| faire | **fer-** | je ferai | I will do / make |
| pouvoir | **pourr-** | je pourrai | I will be able to |
| vouloir | **voudr-** | je voudrai | I will want |
| devoir | **devr-** | je devrai | I will have to |
| venir | **viendr-** | je viendrai | I will come |
| savoir | **saur-** | je saurai | I will know |
| voir | **verr-** | je verrai | I will see |
| envoyer | **enverr-** | j'enverrai | I will send |
| tenir | **tiendr-** | je tiendrai | I will hold |
| courir | **courr-** | je courrai | I will run |
| mourir | **mourr-** | je mourrai | I will die |
| recevoir | **recevr-** | je recevrai | I will receive |
| falloir | **faudr-** | il faudra | it will be necessary |
| pleuvoir | **pleuvr-** | il pleuvra | it will rain |
| valoir | **vaudr-** | il vaudra | it will be worth |

> 💡 **Exam tip:** Irregular future stems = the same stems for the conditional. Learn them once — use for both tenses!

## 7.4 Futur Proche vs Futur Simple

| | Futur Proche | Futur Simple |
|-|-------------|-------------|
| **Formation** | *aller* (present) + infinitive | infinitive + future endings |
| **Example** | *Je vais partir.* | *Je partirai.* |
| **Use** | Immediate / very near future; plans already decided | More distant / formal future; predictions |
| **Register** | Common in spoken French | Common in written French |

**Examples:**
- *Ce soir, je vais regarder un film.* — Tonight, I'm going to watch a film. (planned, near)
- *Un jour, je voyagerai dans le monde entier.* — One day, I'll travel the whole world. (distant, uncertain)

# PART B: Le Conditionnel Présent

## 7.5 When to Use Le Conditionnel

| Use | Example | English |
|-----|---------|---------|
| Conditional sentences | *Si j'avais le temps, je lirais plus.* | If I had the time, I would read more. |
| Polite requests | *Je voudrais un café.* | I would like a coffee. |
| Reported speech (future → conditional) | *Il a dit qu'il viendrait.* | He said he would come. |
| Hypothetical situations | *Avec un million d'euros, j'achèterais une maison.* | With a million euros, I would buy a house. |
| Speculation / uncertain information | *Ce serait possible.* | That might be possible. |
| Wishes and dreams | *J'aimerais devenir médecin.* | I would like to become a doctor. |

---

## 7.6 Formation of Le Conditionnel Présent

**Formula:** Same irregular stems as future + imperfect endings

### Conditional Endings (= Imperfect Endings)

| Pronoun | Ending |
|---------|--------|
| je | **-ais** |
| tu | **-ais** |
| il / elle / on | **-ait** |
| nous | **-ions** |
| vous | **-iez** |
| ils / elles | **-aient** |

### parler — conditionnel

| je parlerais | nous parlerions |
|-------------|----------------|
| tu parlerais | vous parleriez |
| il / elle / on parlerait | ils / elles parleraient |

### Key Irregular Conditionals

| Verb | Conditionnel (je) | English |
|------|------------------|---------|
| être | je **serais** | I would be |
| avoir | j'**aurais** | I would have |
| aller | j'**irais** | I would go |
| faire | je **ferais** | I would do |
| pouvoir | je **pourrais** | I could / would be able to |
| vouloir | je **voudrais** | I would like / want |
| devoir | je **devrais** | I should / ought to |
| venir | je **viendrais** | I would come |
| savoir | je **saurais** | I would know |
| voir | je **verrais** | I would see |

## 7.7 Si-Clauses (Conditional Sentences)

Three types of conditional sentences — each with a fixed tense combination:

| Type | Si clause | Main clause | Example |
|------|-----------|-------------|---------|
| **Real / likely** | Si + **présent** | **Futur simple** | *Si tu étudies, tu **réussiras**.* |
| **Hypothetical** | Si + **imparfait** | **Conditionnel présent** | *Si j'avais le temps, j'**apprendrais** le piano.* |
| **Impossible** | Si + **plus-que-parfait** | **Conditionnel passé** | *Si j'avais su, je **n'aurais pas accepté**.* |

> [!note] NEVER use the future or conditional directly after **si** (in if-clauses). This is a golden rule:
> - ✗ *Si je **serai** riche...* — WRONG
> - ✓ *Si j'**étais** riche...* — CORRECT (hypothetical)

**More examples:**

*Real:*
- *S'il **fait** beau demain, nous **irons** à la plage.* (If the weather is good tomorrow, we'll go to the beach.)

*Hypothetical:*
- *Si j'**étais** riche, j'**achèterais** une grande maison.* (If I were rich, I would buy a big house.)
- *Si on **pouvait** voler, on **voyagerait** partout.* (If we could fly, we would travel everywhere.)

*Impossible:*
- *Si j'**avais** plus travaillé, j'**aurais réussi** l'examen.* (If I had worked harder, I would have passed the exam.)

---

## 7.8 Polite Uses of the Conditional

The conditional is essential for expressing polite requests and offers:

| French | English |
|--------|---------|
| *Je **voudrais** ...* | I would like ... |
| *J'**aimerais** ...* | I would love to ... |
| *Je **pourrais** ...?* | Could I ...? |
| ***Pourriez-vous** ...?* | Could you ...? |
| ***Auriez-vous** ...?* | Would you have ...? |
| *Ce **serait** possible de ...?* | Would it be possible to ...? |
| *Il **faudrait** ...* | It would be necessary to ... |
| *Je **devrais** peut-être ...* | I should perhaps ... |

---

## 7.9 Conditionnel Passé (Would Have)

**Formation:** Conditional of *avoir* or *être* + past participle

| French | English |
|--------|---------|
| *J'**aurais aimé** y aller.* | I would have liked to go. |
| *Elle **serait venue** si elle avait pu.* | She would have come if she could have. |
| *Nous **aurions fait** mieux.* | We would have done better. |
| *Ils **seraient partis** plus tôt.* | They would have left earlier. |

Used in **Type 3 (impossible) si-clauses** — see 7.7 above.

## Key Definitions

| Term | Definition |
|------|-----------|
| Futur simple | Simple future — formed by adding endings to the infinitive (irregular stems exist) |
| Futur proche | Near future — *aller* (present) + infinitive; used for immediate/planned future |
| Conditionnel présent | Present conditional — "would do" — same stems as future + imperfect endings |
| Conditionnel passé | Past conditional — "would have done" — conditional of avoir/être + past participle |
| Phrase conditionnelle | Conditional sentence — *si* + one tense → main clause in another tense |
| Radical du futur | Future stem — the infinitive for most verbs; irregular for être, avoir, aller, faire, etc. |
| Concordance des temps | Sequence of tenses — rules governing which tenses must be used together |
