## 2.1 Subject Pronouns

Subject pronouns replace the subject of a verb.

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | **je** (I) | **nous** (we) |
| 2nd | **tu** (you — informal) | **vous** (you — formal / plural) |
| 3rd masc. | **il** (he / it) | **ils** (they — masc. or mixed group) |
| 3rd fem. | **elle** (she / it) | **elles** (they — feminine only) |
| Impersonal | **on** (one / we / they) | — |

### Tu vs Vous — Register

| Tu | Vous |
|----|------|
| Family, friends, children, animals | Strangers, older people, teachers, formal situations |
| Peers, classmates | Addressing more than one person |

> [!note] Using *tu* with a stranger or authority figure is considered rude in French. When writing formal letters or emails in the Writing paper, always use *vous*.

### On — Multiple Uses

*On* literally means "one" but is commonly used as:
1. **Informal *nous*:** *On va au cinéma.* = *Nous allons au cinéma.* (We're going to the cinema.)
2. **General "you/one":** *On ne sait jamais.* (You never know / One never knows.)
3. **Passive sense:** *On parle français ici.* (French is spoken here.)

*On* **always takes the il/elle form** of the verb: *On mange*, *On est arrivé(s)*.

## 2.2 Direct Object Pronouns (COD)

Direct object pronouns replace nouns that receive the action of the verb directly (no preposition needed).

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | **me** (m') | **nous** |
| 2nd | **te** (t') | **vous** |
| 3rd masc. | **le** (l') | **les** |
| 3rd fem. | **la** (l') | **les** |

### Position

DOPs come **before the conjugated verb**:
- *Je **le** vois.* — I see him / it.
- *Elle **m'**appelle.* — She calls me.
- *Tu **les** aimes ?* — Do you like them?

In **compound tenses**, the DOP goes before the **auxiliary**:
- *Je **l'**ai vu.* — I saw him / it.
- *Il **nous** a appelés.* — He called us.

In **infinitive constructions**, the DOP goes before the **infinitive**:
- *Je veux **le** faire.* — I want to do it.
- *Elle peut **les** aider.* — She can help them.

In **negative sentences**, the DOP stays between *ne* and the verb:
- *Je **ne le** vois **pas**.* — I don't see him.
- *Il **ne les** a **pas** vus.* — He didn't see them.

## 2.3 Indirect Object Pronouns (COI)

Indirect object pronouns replace nouns preceded by **à** (to / for a person).

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | **me** (m') | **nous** |
| 2nd | **te** (t') | **vous** |
| 3rd | **lui** (him or her) | **leur** (them) |

**Examples:**
- *Je **lui** parle.* — I talk to him / her.
- *Elle **nous** écrit.* — She writes to us.
- *Il **leur** téléphone.* — He calls them.

### DOPs vs IOPs — How to Choose

| Use DOP | Use IOP |
|---------|---------|
| No preposition between verb and noun | Preposition *à* between verb and noun |
| *Je vois **Marie**.* → *Je **la** vois.* | *Je parle **à Marie**.* → *Je **lui** parle.* |

**Common verbs taking indirect objects (+ à):**
*parler à, téléphoner à, écrire à, donner à, envoyer à, répondre à, dire à, demander à, offrir à, promettre à, ressembler à*

> [!note] *Lui* and *leur* replace both masculine and feminine: *Je lui parle* = I talk to him OR to her. Context must make this clear.

## 2.4 Reflexive Pronouns

Used with **pronominal (reflexive) verbs**, the reflexive pronoun refers back to the subject.

| Person | Reflexive pronoun |
|--------|------------------|
| je | **me** (m') |
| tu | **te** (t') |
| il / elle / on | **se** (s') |
| nous | **nous** |
| vous | **vous** |
| ils / elles | **se** (s') |

**Full conjugation: *se lever* (to get up) — present tense:**

| Pronoun | Form |
|---------|------|
| je | **me** lève |
| tu | **te** lèves |
| il / elle / on | **se** lève |
| nous | **nous** levons |
| vous | **vous** levez |
| ils / elles | **se** lèvent |

**Common pronominal verbs:**

| Verb | English |
|------|---------|
| se lever | to get up |
| se coucher | to go to bed |
| se réveiller | to wake up |
| se laver | to wash (oneself) |
| se brosser les dents | to brush one's teeth |
| s'habiller | to get dressed |
| se déshabiller | to get undressed |
| se dépêcher | to hurry |
| s'appeler | to be called |
| se souvenir de | to remember |
| s'amuser | to have fun / enjoy oneself |
| se reposer | to rest |
| se sentir | to feel |
| se disputer | to argue |
| se réconcilier | to make up / reconcile |
| s'entendre avec | to get along with |
| se passer | to happen |
| s'inquiéter | to worry |

## 2.5 Stressed / Emphatic Pronouns (Pronoms Toniques)

These are used after prepositions, for emphasis, after *c'est/ce sont*, and in comparisons.

| Person | Stressed pronoun |
|--------|-----------------|
| je → | **moi** |
| tu → | **toi** |
| il → | **lui** |
| elle → | **elle** |
| on → | **soi** |
| nous → | **nous** |
| vous → | **vous** |
| ils → | **eux** |
| elles → | **elles** |

**Uses:**

1. **After prepositions:** *avec **moi*** (with me), *chez **lui*** (at his place), *sans **eux*** (without them), *pour **elle*** (for her)
2. **Emphasis:** ***Moi**, je n'aime pas ça.* (As for me, I don't like that.)
3. **After *c'est/ce sont*:** *C'est **lui**.* (It's him.) / *Ce sont **eux**.* (It's them.)
4. **In comparisons:** *Il est plus grand que **moi**.* (He is taller than me.)
5. **Compound subjects:** ***Lui** et moi, nous partons.* (He and I are leaving.)
6. **On its own:** *— Qui a fait ça ? — **Moi!*** (Who did that? — Me!)

## 2.6 Relative Pronouns

Relative pronouns introduce a relative clause that gives more information about a noun.

| Pronoun | Function | Equivalent |
|---------|----------|-----------|
| **qui** | Subject of the relative clause | who, which, that |
| **que / qu'** | Direct object of the relative clause | whom, which, that |
| **dont** | Replaces *de* + noun | whose, of which, about which |
| **où** | Replaces a place or time expression | where, when |

**Examples:**

- *Le garçon **qui** parle est mon ami.* — The boy **who** is speaking is my friend. (*qui* is subject of *parle*)
- *Le film **que** j'ai vu était super.* — The film **that** I watched was great. (*que* is object of *j'ai vu*)
- *Le professeur **dont** je te parle est sympa.* — The teacher **I'm talking to you about** is nice. (*dont* replaces *de ce professeur* after *parler de*)
- *La ville **où** j'habite est belle.* — The city **where** I live is beautiful. (*où* replaces *dans cette ville*)
- *Le jour **où** je suis né(e)* — The day **when** I was born.

> [!note] *Qui* NEVER elides (never becomes *qu'*). *Que* DOES elide before a vowel: *le film **qu'**il a vu*.

> [!note] With *dont*, the word order is: antecedent + dont + subject + verb + (rest). The object that follows *dont* does NOT take *de*: *l'histoire **dont** il se souvient* (the story he remembers — *se souvenir de*).

### Lequel / Laquelle / Lesquels / Lesquelles

Used after prepositions (instead of *qui/que*):

| | Masculine | Feminine |
|-|-----------|----------|
| Singular | **lequel** | **laquelle** |
| Plural | **lesquels** | **lesquelles** |

Note contractions: *à + lequel = auquel*; *de + lequel = duquel*; *à + lesquels = auxquels*

- *La table **sur laquelle** je travaille.* — The table on which I work.
- *Le problème **auquel** je pense.* — The problem I'm thinking about.

## 2.7 Demonstrative Pronouns

Used to say "this one", "that one", "these", "those".

| | Masculine | Feminine |
|-|-----------|----------|
| Singular | **celui** | **celle** |
| Plural | **ceux** | **celles** |

Add **-ci** (this one / these — near) or **-là** (that one / those — far):
- *Quel livre veux-tu ? — **Celui-ci**.* — Which book do you want? — This one.
- *Je préfère **celle-là**.* — I prefer that one.

Followed by a **relative clause**:
- ***Ceux qui** arrivent en retard devront attendre.* — Those who arrive late will have to wait.
- ***Celle que** tu as achetée est jolie.* — The one you bought is pretty.

Followed by ***de*** to indicate possession:
- *Mon livre et **celui de** Marie.* — My book and Marie's.

## 2.8 Possessive Pronouns

Replace a noun already mentioned, indicating who it belongs to. They agree with the noun replaced (not the possessor).

| | Masc. Sing. | Fem. Sing. | Masc. Plural | Fem. Plural |
|-|------------|-----------|-------------|------------|
| mine | **le mien** | **la mienne** | **les miens** | **les miennes** |
| yours (tu) | **le tien** | **la tienne** | **les tiens** | **les tiennes** |
| his / hers / its | **le sien** | **la sienne** | **les siens** | **les siennes** |
| ours | **le nôtre** | **la nôtre** | **les nôtres** | **les nôtres** |
| yours (vous) | **le vôtre** | **la vôtre** | **les vôtres** | **les vôtres** |
| theirs | **le leur** | **la leur** | **les leurs** | **les leurs** |

**Examples:**
- *Voici mon stylo. Où est **le tien** ?* — Here's my pen. Where is yours?
- *Ma maison est grande ; **la sienne** est petite.* — My house is big; his/hers is small.
- *Nos parents et **les leurs**.* — Our parents and theirs.

> [!note] Agreement is with the **noun replaced**, not the possessor: *sa voiture* → ***la sienne*** (agrees with *voiture*, feminine) — regardless of whether the owner is male or female.

## 2.9 The Pronouns Y and EN

### Y — Replaces à + place / thing (NOT a person)

- Replaces a place: *Je vais **à Paris**.* → *J'**y** vais.* (I'm going there.)
- Replaces *à* + noun (thing): *Il pense **à son avenir**.* → *Il **y** pense.* (He thinks about it.)
- Replaces *à* + verb phrase: *Tu as répondu **à la question** ?* → *Tu **y** as répondu ?*

> [!note] For **people**, use a stressed pronoun, not *y*: *Je pense **à Marie**.* → *Je pense **à elle**.* (NOT *j'y pense*)

### EN — Replaces de + noun / partitive article / noun with number

- Replaces partitive (du/de la/des): *Je mange **des pommes**.* → *J'**en** mange.* (I eat some.)
- Replaces *de* + place: *Il vient **de Paris**.* → *Il **en** vient.* (He comes from there.)
- Replaces noun after a number: *Elle a **deux** chats.* → *Elle **en** a **deux**.* (She has two.)
- Replaces *de* + noun after verbs/expressions: *Tu as besoin **d'aide** ?* → *Tu **en** as besoin ?*

> 💡 **Exam tip:** When *en* replaces a noun with a quantity word or number, the quantity/number stays at the end: *J'en ai trois.* (I have three of them.) *Elle en veut beaucoup.* (She wants a lot of it/them.)

## 2.10 Pronoun Order (When Multiple Pronouns Are Used)

When two pronouns appear in the same clause, they follow a **fixed order** before the verb:

| Position | 1 | 2 | 3 | 4 | 5 |
|----------|---|---|---|---|---|
| Pronouns | me / te / se / nous / vous | le / la / les | lui / leur | y | en |

**Examples:**
- *Il **me le** donne.* — He gives it to me. (me + le)
- *Elle **le lui** envoie.* — She sends it to him. (le + lui)
- *Tu **nous en** parles.* — You talk to us about it. (nous + en)
- *Je **l'y** ai vu.* — I saw him there. (l' + y)

### After an Affirmative Imperative

After an affirmative imperative, pronouns follow the verb with hyphens. Order changes: le/la/les come first, then the person pronouns; *me/te* become **moi/toi**:

| After Imperative | 1 | 2 | 3 |
|-----------------|---|---|---|
| Order | le / la / les | moi / toi / lui / nous / vous / leur | y / en |

- *Donne-**le-moi** !* — Give it to me!
- *Explique-**le-lui**.* — Explain it to him/her.
- *Dis-**lui-en** un peu.* — Tell him/her a little about it.

> [!note] In **negative** imperatives, normal pre-verb order is restored: *Ne **me le** donne pas!* — Don't give it to me!

## Key Definitions

| Term | Definition |
|------|-----------|
| Pronom sujet | Subject pronoun — replaces the grammatical subject (je, tu, il, elle, on, nous, vous, ils, elles) |
| COD (Complément d'Objet Direct) | Direct object pronoun — replaces a noun with no preposition (me, te, le, la, nous, vous, les) |
| COI (Complément d'Objet Indirect) | Indirect object pronoun — replaces *à + person* (me, te, lui, nous, vous, leur) |
| Pronom réfléchi | Reflexive pronoun — used with pronominal verbs to refer back to the subject |
| Pronom tonique | Stressed pronoun — used after prepositions, for emphasis, after c'est (moi, toi, lui, elle...) |
| Pronom relatif | Relative pronoun — introduces a relative clause (qui, que, dont, où) |
| Y | Pronoun replacing *à* + place/thing (= there / about it / to it) |
| EN | Pronoun replacing *de* + noun or a partitive (= some / of it / from there) |
| Pronom démonstratif | Demonstrative pronoun (celui, celle, ceux, celles + -ci/-là) |
| Pronom possessif | Possessive pronoun (le mien, la tienne, les nôtres...) — agrees with noun replaced |
